
### **无穷级数**
1. **常数项级数的概念与性质**
    * **级数收敛的必要条件**
        如果级数 $\sum\limits _ { n = 1 } ^ { \infty } u _ { n } $ 收敛，则它的一般项趋于零，$\lim\limits _ { n \rightarrow \infty } u _ { n } \rightarrow 0 $
        若级数的一般项不趋于零，则级数必发散，但是 $\lim \limits _ { n \rightarrow \infty } u_{n} = 0$ 并非级数收敛的充分条件
