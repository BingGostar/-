### 极限
1. **概念**
    这里只展示几个重要的极限公式

    >$$\lim _{x \rightarrow \infty} \frac{\sin x}{x}=0$$

    >$$\lim _{x \rightarrow 0} \frac{\sin x}{x}=1 (参考夹逼定理)$$

    >$$\lim _{u \rightarrow \infty}\left(1+\frac{1}{n}\right)^{n}=e$$

    >$$\lim _{z \rightarrow 0}(1+z)^{\frac{1}{z}}=\lim _{x \rightarrow \infty}\left(1+\frac{1}{x}\right)^{x}=e$$

    >$$\lim _{x \rightarrow \infty}\left(1-\frac{1}{x}\right)^{x}=\lim _{t \rightarrow \infty}\left(1+\frac{1}{t}\right)^{-t}=\lim _{t \rightarrow \infty} \frac{1}{\left(1+\frac{1}{t}\right)^{t}}=\frac{1}{\mathrm{e}}$$

### 函数的连续性与间断点
1. **函数的连续性**
    * **定义**
        函数 y=f(x)，自变量x从$x_{0}$变到$x_{0}$+Δx，函数y相应的从f($x_{0}$)变到f($x_{0}$+Δx)
        >$$\Delta y=f\left(x_{0}+\Delta x\right)-f\left(x_{0}\right)$$
        
        连续性的概念可以这么描述：如果当Δx趋于0时，函数y对应的增量也趋于0
        >$$\lim _{\Delta x \rightarrow 0} \Delta y=0$$
        >$$\lim _{\Delta x \rightarrow 0}\left[f\left(x_{0}+\Delta x\right)-f\left(x_{0}\right)\right]=0$$

        函数f(x)在$x_{0}$连续的充要条件：
        >$$\lim _{x \rightarrow x_{0}} f(x)=f\left(x_{0}\right)$$
    * **左右连续**
        右连续
        >$$\lim _{x \rightarrow x_{0}^{+}} f(x)=f\left(x_{0}+0\right)=f\left(x_{0}\right)$$

        左连续
        >$$\lim _{x \rightarrow x_{0}^{-}} f(x)=f\left(x_{0}-0\right)=f\left(x_{0}\right)$$

        函数y=f(x)在$x_{0}$处左极限等于右极限等于函数值，则称f(x)在$x_{0}$处连续
        >$$\lim _{x \rightarrow x_{0}^{+}} f(x) = \lim _{x \rightarrow x_{0}^{-}} f(x)$$
2. **函数的间断点**
    * **定义**
        若f(x)在$x_{0}$处不连续，则称$x_{0}$为f(x)的间断点

### Lipschitz连续性(李普希兹连续)
[深度学习中的Lipschitz约束：泛化与生成模型](https://www.jiqizhixin.com/articles/2018-10-16-19)
1. **概念**
    有函数 f(x)，如果存在一个常量K，使得对 f(x) 定义域上（可为实数也可以为复数）的任意两个值满足如下条件：

    >$$\left|f\left(x_{1}\right)-f\left(x_{2}\right)\right| \leq\left|x_{1}-x_{2}\right| * K$$
    
    那么称函数 f(x) 满足Lipschitz连续条件，并称K为 f(x) 的Lipschitz常数。
2. **理解**
    上述等价于f的导函数的绝对值不超过k。比如 log(x) 就不是Lipschitz连续，因为它的导函数没有上界。Lipschitz连续条件限制了一个连续函数的最大局部变动幅度（不能超过某一常量）
3. **应用**
    * L约束与泛化
        记输入为 x，输出为 y，模型为 f，模型参数为 w，记为：
        >$$y=f_{w}(x)$$

        很多时候，我们希望得到一个“稳健”的模型。何为稳健？一般来说有两种含义，一是对于参数扰动的稳定性，比如模型变成了$f_{w}+\Delta w(x)$后是否还能达到相近的效果？如果在动力学系统中，还要考虑模型最终是否能恢复到 fw(x)；二是对于输入扰动的稳定性，比如输入从 x 变成了 x+Δx 后，$f_{w}(x+\Delta x)$ 是否能给出相近的预测结果。
        所以，大多数时候我们都希望模型对输入扰动是不敏感的，这通常能提高模型的泛化性能。于是 Lipschitz 提出了一个更具体的约束，那就是存在某个常数 C（它只与参数有关，与输入无关），使得下式恒成立
        >$$\left\|f_{w}\left(x_{1}\right)-f_{w}\left(x_{2}\right)\right\| \leq C(w) \cdot\left\|x_{1}-x_{2}\right\|$$

        也就是说，希望整个模型被一个线性函数“控制”住。这便是 L 约束了。
        换言之，在这里我们认为满足 L 约束的模型才是一个好模型。并且对于具体的模型，我们希望估算出 C(w) 的表达式，并且希望 C(w) 越小越好，越小意味着它对输入扰动越不敏感，泛化性越好。

### 导数与微分
1. **导数**
    设函数y=f(x)在点x0的某个区域有定义，如果$\frac{\Delta y}{\Delta x}$在当Δx->0时极限存在，则称函数y=f(x)在x0可导，这个极限是函数y=f(x)在x0的导数：
    >$$f^{\prime}\left(x_{0}\right)=\lim _{\Delta x \rightarrow 0} \frac{\Delta y}{\Delta x}=\lim _{\Delta x \rightarrow 0} \frac{f\left(x_{0}+\Delta x\right)-f\left(x_{0}\right)}{\Delta x}$$

2. **求导法则与公式**
    * **求导法则**
        >$$[u(x) \pm v(x)]^{\prime}=u^{\prime}(x) \pm v^{\prime}(x)$$

        >$$[u(x) v(x)]^{\prime}=u^{\prime}(x) v(x)+u(x) v^{\prime}(x)$$

        >$$\left[\frac{u(x)}{v(x)}\right]^{\prime}=\frac{u^{\prime}(x) v(x)-u(x) v^{\prime}(x)}{v^{2}(x)}(v(x) \neq 0)$$

    * **反函数求导法则**
        >$$\left[f^{-1}(x)\right]^{\prime}=\frac{1}{f^{\prime}(y)} 或 \quad \frac{d y}{d x}=\frac{1}{\frac{d x}{d y}}$$

    * **复合函数求导法则**
        存在$y=f(u)$在点$u=g(x)$处可导，则导数为
        >$$\frac{\mathrm{d} y}{\mathrm{d} x}=f^{\prime}(u) \cdot g^{\prime}(x) 或 \quad \frac{\mathrm{d} y}{\mathrm{d} x}=\frac{\mathrm{d} y}{\mathrm{d} u} \cdot \frac{\mathrm{d} u}{\mathrm{d} x}$$

    * **导数公式**
        >$( C ) ^ { \prime } = 0 \ \ \ \ \  ( x ^ { \mu } ) ^ { \prime } = \mu x ^ { \mu - 1 }  $
        $ ( \sin x ) ^ { \prime } = \cos x \ \ \ \ \ ( \cos x ) ^ { \prime } = - \sin x  $
        $( \tan x ) ^ { \prime } = \sec^ { 2 } x \ \ \ \ \  ( \cot x ) ^ { \prime } = - \cos ^ { 2 } x  $
        $( \sec x ) ^ { \prime } = \sec x\tan x \ \ \ \ \ ( \csc x ) ^ { \prime } = - \cos x \cot x  $
        $( a ^ { x } ) ^ { \prime } = a ^ { x } \ln a \ \ \ \ \ ( e ^ { x } ) ^ { \prime } = e ^ { x }  $
        $( \log _ { a } x ) ^ { \prime } = \frac { 1 } { x \ln a } \ \ \ \ \ ( \ln x ) ^ { \prime } = \frac { 1 } { x }  $
        $( \arcsin x ) ^ { \prime } = \frac { 1 } { \sqrt { 1 - x ^ { 2 } } } \ \ \ \ \ ( \arccos x ) ^ { \prime } = - \frac { 1 } { \sqrt { 1 - x ^ { 2 } } }  $
        $( \arctan x ) ^ { \prime } = \frac { 1 } { 1 + x ^ { 2 } } \ \ \ \ \ ( \arctan x ) ^ { \prime } = - \frac { 1 } { 1 + x ^ { 2 } }  $
        
    * **莱布尼茨公式**
        莱布尼茨公式用于对两个函数的乘积求取其高阶导数
        一般的，如果函数u=u(x)与函数v=v(x)在点x处都具有n阶导数，那么此时有
        >$$(uv) ^ {(n)} = \sum\limits _ { k = 0 } ^ { n } C _ { n } ^ { k } u ^ { ( n - k ) } v ^ {(k)}$$
        
        注意区分二项式展开公式
        >$$( u + v ) ^ { n } = \sum\limits _ { k = 0 } ^ { n } C _ { n } ^ { k } u ^ { n - k } v ^ { k }$$

    * **隐函数导数**
        函数 $y=f(x)$ 表示显函数
        方程 $F(x,y)=0$ 所确定的y是x的函数称为隐函数

        * **幂指函数**
            对于一般幂指函数 $y = a ^ { x } ( a > 0 ) $，可以采取两边取对数，然后求导
            >$\ln y = x \ln a \ => \ (\ln y)^{\prime} = ( x \ln a)^{\prime}  $

3. **微分**
    设函数y=f(x)在某个领域内有定义，$x_{0}$及$x_{0}+\Delta x$在这区间内，如果增量：
    >$$\Delta y=f\left(x_{0}+x\right)-f\left(x_{0}\right)$$

    可表示为
    >$$\Delta y=A \Delta x+o(\Delta x)$$

    其中A是不依赖$\Delta x$的常数，$o(\Delta x)$是指$\Delta x$趋于0时的高阶无穷小，那么称函数y=f(x)在点$x_{0}$是可微的，而$A\Delta x$叫做函数在点$x_{0}$相应于自变量增量$\Delta x$的微分，记作$\mathrm{d} y$
    >$$\mathrm{d} y=A \Delta x$$

4. **微分与导数的关系**
    如果函数f(x)在点$x_{0}$可微，则f(x)在点$x_{0}$也一定可导，且$A=f^{\prime}\left(x_{0}\right)$。反之，如果f(x)可导，存在下式：
    >$$\lim _{\Delta x \rightarrow 0} \frac{\Delta y}{\Delta x}=f^{\prime}\left(x_{0}\right)$$

    根据极限与无穷小的关系转化上式，当$\Delta x \rightarrow 0$时
    >$$\frac{\Delta y}{\Delta x}=f^{\prime}\left(x_{0}\right)+\alpha$$

    其中当$\Delta x \rightarrow 0$时，$\alpha$无穷小，所以
    >$$\Delta y=f^{\prime}\left(x_{0}\right) \Delta x+o(\Delta x)$$

    函数f(x)在点$x_{0}$可微的充分必要条件是函数f(x)在点$x_{0}$可导
    >$$\mathrm{d} y=f^{\prime}\left(x_{0}\right) \Delta x$$

    函数的微分表达式
    >$$\mathrm{d} y = f ^ { \prime } ( x ) \mathrm{d} x  $$

6. **微分的应用**
    * **近似计算**
        >$$f ( x ) \approx f ( x _ { 0 } ) + f ^ { \prime } ( x _ { 0 } ) ( x - x _ { 0 } )  $$

### 微分中值定理
1. **罗尔中值定理**
    如果函数$y=f(x)$满足下列条件：
    1) 在闭区间[a,b]上连续
    2) 在开区间(a,b)上可导
    3) f(a) = f(b)

    则开区间(a,b)至少存在一点$\xi$，使得
    >$$f^{\prime}(\xi)=0$$

2. **拉格朗日中值定理**
    如果函数$y=f(x)$满足下列条件：
    1) 在闭区间[a,b]上连续
    2) 在开区间(a,b)上可导

    则开区间(a,b)至少存在一点$\xi$，使得
    >$$f(b)-f(a)=f^{\prime}(\xi)(b-a)$$

    存在$x_{0}, x_{0} + \Delta x\in (a,b)$，则有
    >$$f ( x _ { 0 } + \Delta x ) - f ( x _ { 0 } ) = f ^ { \prime } ( x _ { 0 } + \theta \Delta x ) \cdot \Delta x \ \ \ \ ( 0 < \theta < 1 )  $$
    
   此公式又称有限增量公式或有限增量定理，它与微分公式的近似有区别
   >$$f ( x + \Delta x ) \approx f ( x ) + f ^ { \prime } ( x ) \Delta x \ \ \ (微分，近似)$$

   >$$f ( x + \Delta x ) = f ( x ) + f ^ { \prime } ( x + \theta A x ) \Delta x \ \ \ (拉氏，精确)$$

3. **柯西中值定理**
    如果函数$f(x)$和$g(x)$满足
    1) 在闭区间[a,b]上连续
    2) 在开区间(a,b)上可导
    3) 对任意$x \in ( a , b )  $，$g ^ { \prime } ( x ) \neq 0  $

    则存在
    >$$\frac { f ( b ) - f ( a ) } { g ( b ) - g ( a ) } = \frac { f ^ { \prime } ( \xi ) } { g^{\prime} ( \xi ) }  $$

### 导数的应用
1. **洛必达法则**
    用来求极限的方法，在计算$0/0$ 或 $\infty/\infty $的极限时，可用下式：
    >$$\lim\limits _ { x \rightarrow a } \frac { f ( x ) } { F ( x ) } = \lim\limits _ { x \rightarrow a } \frac { f ^ { \prime } ( x ) } { F ^ { \prime } ( x ) }  $$

2. **泰勒公式**
    用多项式函数去逼近光滑函数([通俗理解泰勒公式](https://www.zhihu.com/question/21149770))
    设$n$是一个正整数，如果定义在一个包含 $a$ 区间上的函数 $f$ 在 $a$ 点处 $n+1$ 次可导，那么对于这个区间上的任意$x$都有：
    >$$f ( x ) = \sum\limits _ { n = 0 } ^ { N } \frac { f ^ { ( n ) } ( a ) } { n ! } ( x - a ) ^ { n } + R _ { n } ( x )  $$

    其中多项式部分称为函数在 $a$ 处的泰勒展开式。$R _ { n } ( x )  $ 是泰勒公式的余项且是 $( x - a ) ^ { n }  $ 的高阶无穷小
    如果$a=0$, 就是麦克劳伦公式
    >$$f ( x ) = \sum\limits _ { n = 0 } ^ { N } \frac { f ^ { \prime } ( 0 ) } { n ! } x ^ { n } + R _ { n } ( x )  $$

    泰勒公式的余项代数式如下，其中$a < \theta < x  $
    >$$R _ { n } ( x ) = \frac { f ^ { ( n + 1 ) } ( \theta ) } { ( n + 1 ) ! } ( x - a ) ^ { ( n + 1 ) }  $$

    当 $n=0$ 时，根据泰勒公式得到
    >$$f ( x ) = f ( a ) + f ^ { \prime } ( \theta ) ( x - a )  $$

    所以拉格朗日中值定理根本就是 $n=0$ 时的泰勒公式

    略

### 不定积分及其计算
1. **概念与性质**
    在区间 $I$ 上定义两个函数，且满足 $F^{\prime}(x)=f(x)$ 或 $dF(x)=f(x)dx$，则称 $F(x)$ 为 ${f(x)}$ 在 ${I}$ 区间上的原函数。表示为
    >$$\int f(x)dx=F(x)+C$$

    积分的性质
    >$$\int k f(x) d x = k \int f( x ) d x \quad ( k \neq 0 ) $$
    
    >$$\int \lbrack f ( x ) \pm g ( x ) \rbrack d x = \int f ( x ) d x \pm \int g ( x ) d x  $$
    
    >$$\lbrack \int f ( x ) d x + \int g ( x ) d x \rbrack^{\prime} = \lbrack \int f ( x ) d x \rbrack ^{\prime} + \lbrack \int g ( x ) d x \rbrack ^{\prime}$$

2. **基本积分表**
    >$\int k dx = k x + C $
    $\int x ^ { u } d x = \frac { x ^ { u + 1 } } { u + 1 } + C  $
    $\int \frac { 1 } { x } d x = \ln | x | + C  $
    $\int \frac { 1 } { 1 + x ^ { 2 } } d x = \arctan x + C = - a r c \cot x + C  $
    $\int \frac { 1 } { \sqrt { 1 - x ^ { 2 } } } = \arcsin x + C = - \arccos x + C  $
    $\int \cos x d x = \sin x + C  $
    $\int \sin x d x = - \cos x + C  $
    $\int \frac{1}{\cos^{2}x}dx = \int \sec^{2}xdx = \tan x + C  $
    $\int \frac{1}{\sin^{2}x}dx = \int \cos^{2}xdx = - \cot x + C  $
    $\int \sec x \tan x dx = \sec x + C$
    $\int e^{x} dx = e^{x} + C$
    $\int a^{x} dx = \frac {a^{x}} {\ln a} + C$

3. **计算方法**
    * **换元积分法**
        设 $f(u)$ 具有原函数 $F(u)$，即 $F^{ \prime }( u )=f(u)$，$\int f ( u ) d u = F ( u ) + C  $
        如果 $u = \varphi(x)$，且设 $\varphi(x)$ 可微，根据复合函数微分法
        >$$d F \lbrack \varphi ( x ) \rbrack = f \lbrack \varphi ( x ) \rbrack \varphi ^ { \prime } ( x ) d x  $$
    
        其不定积分为
        >$$\int f \lbrack \varphi ( x ) \rbrack \varphi ^ { \prime } ( x ) d x  = F \lbrack \varphi ( x ) \rbrack + C = \lbrack \int f ( u ) d u \rbrack _ { u = \varphi (x) }   $$
    
    * **分部积分法**
        两个函数的乘积导数为

        >$$( u v ) ^ { \prime } = u ^ { \prime } v + u v ^ { \prime }    =>  uv^{\prime}=(uv)^{\prime}-u^{\prime}v$$

        两边进行不定积分
        >$$\int uv^{ \prime } d x = uv - \int u^{\prime} vdx$$

        >$$\int udv = uv - \int vdu$$


### **定积分的计算及其应用**
1. **概念**
    函数 $f(x)$ 在区间 $[a,b]$ 上有界，在区间 $[a,b]$ 内插入 $n-1$ 个点: $a=x_{0}<x_{1}<x_{2}<...<x_{n-1} < x_{n} = b$，记 $\Delta x=x_{i}-x_{i-1}(i=1,\cdots,n)$，$\lambda = \max \lbrace \Delta x _ { 1 } , \Delta x _ { 2 } , \cdots , \Delta x _ { n } \rbrace$，在小区间 $[x_{i-1},x_{i}]$上任取一点，$\xi _ { i } ( i = 1 , 2 , \cdots , n )$，做和 $\sum\limits _ { i = 1 } ^ { n } f ( \xi _ { i } ) \Delta x _ { i } $；如果当 $\lambda \rightarrow 0  $时，上述的和存在极限，且极限值与区间[a,b]的分法和 $\xi_{i}$ 的取法无关，则称此极限为函数 $f(x)$的定积分：
    >$$\int _ { a } ^ { b } f ( x ) d x = \lim\limits _ { \lambda \rightarrow 0 }\sum\limits_{i=1}^{n}f(\xi_{i}) \Delta x_{i}$$
    
    此时称 $f(x)$ 在 $[a,b]$ 上可积
    注意：定积分表示一个数，而不定积分是一个函数族，分别对定积分和不定积分求导
    >$$\frac { d } { d x } \int f ( x ) d x = f ( x ) \ \ \ \ \ \ \quad \frac { d } { d x } \int _ { a } ^ { b } f ( x ) d x = 0  $$

2. **性质**
    >$$\int _ { a } ^ { b } \lbrack f ( x ) \pm g ( x ) \rbrack d x = \int _ { a } ^ { b } f ( x ) d x \pm \int _ { a } ^ { b } g ( x ) d x $$

    >$$\int _ { a } ^ { b } k f ( x ) d x = k \int _ { a } ^ { b } f ( x ) d x $$

    >$$\int _ { a } ^ { b } f ( x ) d x = \int _ { a } ^ { c } f ( x ) d x + \int _ { e } ^ { b } f ( x ) d x \ \ (a < c < b)$$
 
    设M和m分别是f(x)在区间[a,b]上的最大值和最小值，则
    >$$m ( b - a ) \leq \int _ { a } ^ { b } f ( x ) d x \leq M ( b - a )  $$

3. **定积分中值定理**
    若f(x)在闭区间上[a,b]连续,则积分区间[a,b]上至少存在一个点$\xi$，使
    >$$\int _ { a } ^ { b } f ( x ) d x = f ( \xi ) ( b - a ) \ \ \quad ( a \leq \xi \leq b )  $$

4. **牛顿-莱布尼兹公式**
    如果函数 $F(x)$ 是连续函数 $f(x)$ 在区间 $[a,b]$ 上的一个原函数
    >$$\int _ { a } ^ { b } f (x) dx= F(b)-F(a)$$

    揭示了定积分与被积函数的原函数或不定积分之间的关系

5. **反常积分(广义积分)**
    * **无穷限的反常积分**
        设函数 $f(x)$ 在区间 $\lbrack a, +\infty)$ 上连续，取 $t>a$，其广义积分为
        >$$\int _ { a } ^ { +\infty } f ( x ) d x = \lim\limits _ { t \rightarrow + \infty } \int _ { a } ^ { t } f ( x ) d x  $$

        如果极限 $\lim\limits _ { t \rightarrow + \infty } \int _ { a } ^ { t } f ( x ) d x$ 存在，则称广义积分收敛，如果不存在，则称广义积分发散

        类似在区间 $(-\infty, b \rbrack$ 上连续，其广义积分为
        >$$\int _ { -\infty } ^ { b } f ( x ) d x = \lim\limits _ { t \rightarrow - \infty } \int _ { t } ^ { b } f ( x ) d x  $$

        同理，如果极限 $\lim\limits _ { t \rightarrow - \infty } \int _ { t } ^ { b } f ( x ) d x  $ 存在，则称广义积分收敛，如果不存在，则称广义积分发散

        如果函数在区间 $(-\infty, +\infty)$ 上连续，其广义积分为
        >$$\int _ { - \infty } ^ { \infty } f ( x ) d x = \int _ { - \infty } ^ { o } f ( x ) d x + \int _ { 0 } ^ { \infty } f ( x ) d x = \lim\limits _ { a \rightarrow - \infty } \int _ { a } ^ { 0 } f ( x ) d x + \lim\limits _ { b \rightarrow \infty } \int _ { 0 } ^ { b } f ( x ) d x  $$

        同理，如果极限存在，则称广义积分收敛，如果不存在，则称广义积分发散

    * **无界函数的反常积分**
        如果函数在点 $a$ 的任一邻域内都无界，那么点 $a$ 称 $f(x)$ 函数的瑕点（也称无界间断点）
        设 $f(x)$ 在 $(a,b]$ 上连续，则其反常积分如下。如果极限存在，则称反常积分收敛。不存在，则称反常积分发散
        >$$\int _ { a } ^ { b } f ( x ) d x = \lim\limits _ { t \rightarrow a } \int _ { t } ^ { b } f ( x ) d x  $$

        计算无界函数的反常积分，可用牛顿-莱布尼兹公式。那么 $f(x)$ 在 $(a,b]$ 上的反常积分为
        >$$\int _ { a } ^ { b } f ( x ) d x = F ( b ) - \lim\limits _ { x \rightarrow a^{+} } F ( x ) = F (b)-F (a^{+})$$

        如果极限 $\lim\limits _ { x \rightarrow a^{+} } F ( x )$ 存在，则称反常积分收敛，否则发散

    * **反常积分的审敛法**
        我们通常对反常积分是发散还是收敛很感兴趣，然而计算极限往往令人沮丧，幸而我们了解增长和衰减速率，将被积函数替换成更快或更慢的函数，以此判断反常积分的收敛性，这种方法就是审敛法。

        * **无穷限反常积分审敛法**
            * **比较审敛法1**
                设函数 $f(x)$ 在区间 $[a, +\infty)$ $(a > 0)$ 上连续，且 $f ( x ) \geq 0$

                如果存在常数 $M > 0$ 及 $p > 1$，使得 $f ( x ) \leq \frac { M } { x ^ { p } } , ( a \leq x < + \infty ) $，则 $\int _ { a } ^ { + \infty } f ( x ) d x$ 收敛

                如果存在常数 $N>0$，使得 $f ( x ) \geq \frac { N } { x } ( a \leq x < + \infty ) $，则 $\int _ { a } ^ { + \infty } f ( x ) d x$ 发mark散

            * **极限审敛法1**
                设函数 $f(x)$ 在区间 $[a, +\infty)$ $(a > 0)$ 上连续，且 $f ( x ) \geq 0$

                如果存在常数 $p > 1$，使得 $\lim\limits _ { x \rightarrow \infty } x ^ { p } f ( x )$ 存在，则 $\int _ { a } ^ { + \infty } f ( x ) d x$ 收敛

                如果 $\lim\limits _ { x \rightarrow + \infty } x f ( x ) = d > 0 $ (或 $\lim\limits _ { x \rightarrow + \infty } x f ( x ) = + \infty $)，则 $\int _ { a } ^ { + \infty } f ( x ) d x$ 发散
        
        * **无界函数的反常积分审敛法**
            * **比较审敛法2**
                设函数 $f(x)$ 在区间 $(a, b]$ 上连续，且 $f ( x ) \geq 0$，$x$ 是瑕点
                
                如果存在常数 $M > 0$ 及 $q<1$，使得 $f ( x ) \leq \frac { M } { ( x - a ) ^ { q } } ( a < x \leq b )  $，则称反常积分 $\int _ { a } ^ { b } f ( x ) d x $ 收敛

                如果存在常数 $N > 0$ ，使得 $f ( x ) \geq \frac { N } { x - a } \quad ( a < x \leq b )$，则称反常积分 $\int _ { a } ^ { b } f ( x ) d x $ 发散

            * **极限审敛法2**
                设函数 $f(x)$ 在区间 $(a, b]$ 上连续，且 $f ( x ) \geq 0$，$x$ 是瑕点
                
                如果存在常数 $ 0 < q < 1 $，使得 $\lim\limits _ { x \rightarrow a ^ { + } } ( x - a ) ^ { q } f ( x ) $ 存在，则称反常积分 $\int _ { a } ^ { b } f ( x ) d x $ 收敛

                如果 $\lim\limits _ { x \rightarrow a ^ { + } } ( x - a ) f ( x ) = d > 0 $ （或 $\lim\limits _ { x \rightarrow a ^ { + } } ( x - a ) f ( x ) = + \infty $），则称反常积分 $\int _ { a } ^ { b } f ( x ) d x $ 发散

6. **Gamma函数**
    * **函数定义**
        （此函数广义积分的收敛性分析此处略）
        >$$\Gamma ( s ) = \int _ { 0 } ^ { + \infty } e ^ { -x }x^{s-1} d x \quad ( s > 0 ) $$

        函数特点：
        （1）$s-1<0$ 时 $x=0$ 是被积函数的瑕点
        （2）广义积分 $\int _ { 0 } ^ { + \infty } e ^ { -x }x^{s-1} d x $ 对 $s>0$ 均收敛
        $\Gamma $ 函数如图
        ![](https://cn.bing.com/th?id=OIP.DCp106myuC-kyqC3WAVlmwAAAA&pid=Api&rs=1)

    * **函数性质**
        * 递推公式 $\Gamma (s+1)=s\Gamma (s)(s>0)$
            >$\Gamma ( 2 ) = 1 \cdot \Gamma ( 1 ) = 1$
            $\Gamma ( 3 ) = 2 \cdot \Gamma ( 2 ) = 2!$
            $\Gamma ( 4 ) = 3 \cdot \Gamma ( 3 ) = 3!$
            ...
            $\Gamma(n+1) = n!$ 
            把 $\Gamma$ 函数可看成阶乘的推广

        * 当 $s \rightarrow 0^{+}$ 时，$\Gamma(s) \rightarrow +\infty$

        * $\Gamma ( s ) \Gamma ( 1 - s ) = \frac { \pi } { \sin \pi s } \ \ \ ( 0 < s < 1 )$。此公式又称“余元公式”
            当 $s = \frac { 1 } { 2 }$ 时，
            >$\Gamma ( \frac { 1 } { 2 } ) = \sqrt { \pi }$

        * 将 $\Gamma$ 函数进行变换，$x=u^{2}$，则 $\Gamma ( s ) = 2 \int _ { 0 } ^ { \infty } e ^ { - u ^ { 2 } } u ^ { 2 s - 1 } du $，再令 $t=2s-1$，既有 $\int _ { 0 } ^ { \infty } e ^ { - u ^ { 2 } } u ^ { t } d u = \frac { 1 } { 2 } \Gamma ( \frac { 1 + t } { 2 } ) , t > -1$
            令 $s = \frac { 1 } { 2 }$，可得泊松积分
            >$\int _ { - \infty } ^ { \infty } e ^ { - x ^ { 2 } } d x = \sqrt { \pi } $
            $\int _ { 0 } ^ { \infty } e ^ { - x ^ { 2 } } d x = \frac { \sqrt { \pi } } { 2 }  $


### **微分方程（常微分方程）**

* **相关概念**
    含有自变量x、函数y及函数各阶导数的方程称为微分方程，其中函数y是一元函数，导数项必须有，自变量x或函数y可以没有
    >$$ F ( x , y , y ^ { \prime } , \cdots , y ^ { ( n ) } ) = 0 $$

    微分方程中导数的最高项称为微分方程的阶
    满足微分方程函数的解称为微分方程的解，含有独立任意常数与微分方程的阶数相同的解，称为通解
    \
    线性相关与线性不相关
	设 $y_{1}(x),y_{2}(x) ... y_{n}(x)$ 为定义在区间 $I$ 上的 $n$ 个函数，如果存在n个不全为零的的常数 $k_{1},k_{2} ... k_{n}$，使得下式成立，则称这 $n$ 个函数在区间 $I$ 上线性相关，否则线性不相关
    >$$k _ { 1 } y _ { 1 } + k _ { 2 } y _ { 2 } + \cdots + k _ { n } y _ { n } = 0 $$


1. **一阶微分方程的初等解法**
    >$$F ( x , y , y ^ { \prime } ) = 0$$

    * **可分离变量的微分方程**
        如果一个一阶微分方程可写成
        >$$g ( y ) d y = f ( x ) d x $$

        两边积分
        >$$\int g ( y ) d y = \int f ( x ) d x  $$

        $G(y)$ 和 $F(x)$ 分别是 $g(y)$ 和 $f(x)$ 的原函数
        >$$G(y) = F(x) + C$$

    * **齐次方程**
        如果一阶方程可写成如下，那么就称为齐次方程
        >$$\frac { d y } { d x } = \varphi ( \frac { y } { x } )  $$

        对于齐次方程，令 $u = \frac { y } { x }$，$y=xu$，所以$\frac { d y } { d x } = u + x \frac { d u } { d x }  $，代入原方程
        >$$x \frac { d u } { d x } = f ( u ) - u  $$

        分离变量
        >$$\frac {du} {f(u)-u} = \frac {dx}{x}$$

        上述是可分离变的微分方程，求出解后，用 $u= \frac {y}{x}$ 代入即得到通解

    * **一阶线性微分方程**
        标准形式
        >$$\frac { d y } { d x } + P ( x ) y = Q ( x ) $$
            
        >当 $Q(x) = 0$ 时，称为齐次线性方程
        >当 $Q(x) \neq 0$ 时，称为非齐次线性方程

        **一阶齐次线性微分方程通解**
        >$$y = Ce ^ {-\int P (x) dx} \ \ (C = \pm e ^ { c_{1}}) $$

        **一阶非齐次线性微分方程通解**
        >$$y = C e ^ { - \int P ( x ) d x } + e ^ { - \int P ( x ) d x } \int Q ( x ) e ^ { \int P(x)dx }dx $$

        **一阶非齐次线性微分方程的通解 = 齐次方程的通解 + 非齐次方程的一个特解**

        推导过程
        （1）齐次方程
        >分离变量得
        >$\frac { d y } { y } = - P ( x ) d x$
        >两边积分得到通解
        >$\ln |y| = -\int P (x) dx + C_{1}$
        >$y = Ce ^ {-\int P (x) dx} \ \ (C = \pm e ^ { c_{1}}) $

        （2）非齐次方程
        >上述使用**常数变易法**进行求解，此方法把通解中的 $C$ 换成 $x$ 的未知函数 $u(x)$
        >$y = u ( x ) \cdot e ^ { - \int P ( x ) d x } (u(x) \neq 0)$       (1)
        >对 $y$ 进行求导
        >$\frac { d y } { d x } = u ^ { \prime } e ^ {  - \int P(x) dx } - u P ( x ) e ^ { - \int P ( x ) d x}$
        >将 $y$ 和 $\frac {dy}{dx}$ 带入非齐次线性方程中 $\frac { d y } { d x } + P ( x ) y = Q ( x ) $
        >$\Rightarrow  u ^ { \prime } e ^ {- \int P ( x ) d x} - u P ( x ) e ^ { - \int P ( x ) d x } + P ( x ) u e ^ { - \int P(x)dx } = Q ( x )  $
        >$\Rightarrow u ^ { \prime } e ^ { - \int  P ( x ) d x }$
        >$\Rightarrow u^{\prime} = Q(x)e^{\int P(x)dx}$
        >两边积分
        >$\int u^{\prime} du = \int Q ( x ) e ^ { \int P ( x ) d x }dx  $
        >$u = \int Q ( x ) e ^ { \int P ( x ) d x }dx + C $
        >将 $u$ 带入上式(1)
        >$y = C e ^ { - \int P ( x ) d x } + e ^ {-\int P ( x ) d x } \int Q ( x ) e ^ { \int P(x)dx }dx $

3. **高阶线性微分方程**
	* **二阶线性微分方程**
        齐次方程
        >$$y ^ { \prime \prime } + P ( x ) y ^ { \prime } + Q ( x ) y = 0$$
 
        如果 $y_{1}(x)$ 和 $y_{2}(x)$ 是两个线性无关的解（$\frac {y_{1}}{y_{2}} \neq C$），那么通解为
        >$$y = C_{1} y_{1} ( x ) + C _ { 2 } y _ { 2 } ( x ) \ \ \ \ (C_{1}, C_{2}是任意常数) $$
        
        非齐次方程
        >$$y ^ { \prime \prime } + P ( x ) y ^ { \prime } + Q ( x ) y = f(x)$$
        
        设 $y^{*}(x)$ 是二阶非齐次方程的一个特解，$Y(x)$ 是上述齐次方程的通解，那么非齐次方程的通解为
        >$$y = Y(x) + y^{*}(x)$$

	* **高阶线性微分方程**
        齐次方程
    	>$$y ^ { (n) } + a _ { 1 } ( x ) y ^ { (n-1) } + \cdots + a _ { n-1 } ( x ) y ^ { \prime } + a _ { n } ( x ) y = 0  $$

        如果 $y_{1}(x), y_{2}(x) \cdots  y_{n}(x)$ 是n阶齐次方程的n个线性无关的解，那么通解为
        >$$y = C_{1} y_{1} ( x ) + C _ { 2 } y _ { 2 } ( x ) + \cdots + C _ { n } y _ { n } ( x ) \ \ \ \ (C_{1}, C_{2} \cdots C_{n} 是常数) $$

        非齐次方程
        >$$y ^ { n } + a _ { 1 } ( x ) y ^ { n - 1 } + \cdots + a _ { n - 1 } ( x ) y ^ { \prime } + a _ { n } ( x ) y = f ( x )  $$

4. **常系数线性微分方程**
    * **常系数齐次线性微分方程**
        * **二阶常系数齐次微分方程**
            >$$y ^ { \prime \prime } + p y ^ { \prime } + q y = 0 \ \ \ (p, q 常数) $$

            设 $y=e^{rx}$，$r$ 为常数，那么
            >$$y^{\prime} = re^{rx} \ \ \ y^{\prime \prime} = r^{2}e^{rx} $$

            代入微分方程
            >$$(r ^ { 2 } + p r + q)e^{rx} = 0  $$
            
            因为 $e^{rx} \neq 0$，所以
            >$$r ^ { 2 } + p r + q = 0$$

            上式叫做微分方程的**特征方程**，其特征方程的两个根为
            >$$r _ { 1,2 } = \frac { - p \pm \sqrt { p ^ { 2 } - 4 q } } { 2 }  $$

            **通解**
            (1) 两个不相等的实根 $r_{1}, r_{2}$，通解为 $y = C_{1}e^{r_{1}x} + C_{2}e^{r_{2}x}$
            (2) 两个相等的实根 $r_{1} = r_{2}$，通解为 $y = (C_{1}+C_{2}x)e^{r_{1}x}$
            (3) 一对共轭复根 $r_{1,2} = \alpha \pm i \beta $，通解为 $y = e^{\alpha x}(C_{1}\cos \beta x + C_{2}\sin \beta x)$

        * **n阶常系数齐次微分方程**
            >$$y ^ { ( n ) } + p _ { 1 } y ^ { ( n - 1 ) } + \cdots + p _ { n - 1 } y ^ { \prime } + p _ { n } y = f ( x ) \ \ \ \ (p_{1} \cdots p_{n} 都是常数) $$

            用记号 $D$（微分算子）表示对 $x$ 求导的运算 $\frac {d} {dx}$，把 $\frac {dy}{dx}$ 记作 $Dy$，$\frac {d^{n}y}{dx^{n}}$ 记作 $D^{n}y$。
            上述方程可以写作
            >$$( D ^ { n } + p _ { 1 } D ^ { n - 1 } + \cdots + p _ { n - 1 } D + p _ { n } ) y = 0  $$

            >$$L ( D ) = D ^ { n } + p _ { 1 } D ^ { n - 1 } + \cdots + p _ { n - 1 } D + p _ { n }  $$

            L(D) 叫做微分算子 D 的 n 次多项式，可记作
            >$$L(D)y = 0$$

            令 $y=e^{rx}$，由于 $De^{rx} = re^{rx} \dots D^{n}e^{rx} = r^{n} e^{rx}$，所以 $L(D)e^{rx}=L(r)e^{rx}$ 代入上式
            >$$L(r)e^{rx} = 0 \ \Rightarrow \ L(r) = 0$$

            **特征方程**
            >$$r ^ { n } + p _ { 1 } r ^ { n - 1 } + p _ { 2 } r ^ { n - 2 } + \cdots + p _ { n - 1 } r + p_{ n } = 0  $$

            **通解**
            (1) 单实根 $r$，给出一项 $Ce^{rx}$
            (2) 一对单负根 $r_{ 1,2 } = \alpha \pm i \beta$，给出两项 $e^{\alpha x}(C_{1} \cos \beta x + C_{2} \sin \beta x)$
            (3) $k$ 个实根 $r$，给出 $k$ 项 $e ^ { r x} ( C _ { 1 } + C _ { 2 } x + \cdots + C _ { k } x ^ { k - 1 } )  $
            (4) 一对$k$重复根 $k_{1,2} = \alpha \pm i \beta$，给出 $2k$ 项 $e ^ { \alpha r } \lbrack ( C _ { 1 } + c _ { 2 } x + \cdots + C _ { k } x ^ { k - 1 } ) \cos \beta x + (D_{1}+D_{2}x+\dots+D_{k}x^{k-1}) \sin \beta x]  $
            另外，n次代数方程有n个根，每一项各含任意常数，n阶常系数齐次线性微分方程的通解为
            >$$y = C_{1} y _ { 1 } + C _ { 2 } y _ { 2 } + \cdots + C _ { n } y _ { n } $$

            例如：
            $y^{(4)} - 2 y^{\prime \prime \prime } + 5y^{\prime \prime }$ 特征方程 $r ^ { 4 } - 2 r ^ { 3 } + 5 r ^ { 2 } = 0  $ 的根为 $r_{1}=r_{2}=0$ 和 $r_{3,4}=1 \pm 2i$ 通解为
            >$y = C _ { 1 } + C _ { 2 } x + e ^ { x } ( C _ { 3 } \cos 2 x + C _ { 4 } \sin 2 x )  $
