### 函数的连续性与间断点
1. **函数的连续性**
    * **定义**
        函数 y=f(x)，自变量x从$x_{0}$变到$x_{0}$+Δx，函数y相应的从f($x_{0}$)变到f($x_{0}$+Δx)
        >$$\Delta y=f\left(x_{0}+\Delta x\right)-f\left(x_{0}\right)$$
        
        连续性的概念可以这么描述：如果当Δx趋于0时，函数y对应的增量也趋于0
        >$$\lim _{\Delta x \rightarrow 0} \Delta y=0$$
        >$$\lim _{\Delta x \rightarrow 0}\left[f\left(x_{0}+\Delta x\right)-f\left(x_{0}\right)\right]=0$$

        函数f(x)在$x_{0}$连续的充要条件：
        >$$\lim _{x \rightarrow x_{0}} f(x)=f\left(x_{0}\right)$$
    * **左右连续**
        右连续
        >$$\lim _{x \rightarrow x_{0}^{+}} f(x)=f\left(x_{0}+0\right)=f\left(x_{0}\right)$$

        左连续
        >$$\lim _{x \rightarrow x_{0}^{-}} f(x)=f\left(x_{0}-0\right)=f\left(x_{0}\right)$$

        函数y=f(x)在$x_{0}$处左极限等于右极限等于函数值，则称f(x)在$x_{0}$处连续
        >$$\lim _{x \rightarrow x_{0}^{+}} f(x) = \lim _{x \rightarrow x_{0}^{-}} f(x)$$
2. **函数的间断点**
    * **定义**
        若f(x)在$x_{0}$处不连续，则称$x_{0}$为f(x)的间断点

### Lipschitz连续性(李普希兹连续)
[深度学习中的Lipschitz约束：泛化与生成模型](https://www.jiqizhixin.com/articles/2018-10-16-19)
1. **概念**
    有函数 f(x)，如果存在一个常量K，使得对 f(x) 定义域上（可为实数也可以为复数）的任意两个值满足如下条件：

    >$$\left|f\left(x_{1}\right)-f\left(x_{2}\right)\right| \leq\left|x_{1}-x_{2}\right| * K$$
    
    那么称函数 f(x) 满足Lipschitz连续条件，并称K为 f(x) 的Lipschitz常数。
2. **理解**
    上述等价于f的导函数的绝对值不超过k。比如 log(x) 就不是Lipschitz连续，因为它的导函数没有上界。Lipschitz连续条件限制了一个连续函数的最大局部变动幅度（不能超过某一常量）
3. **应用**
    * L约束与泛化
        记输入为 x，输出为 y，模型为 f，模型参数为 w，记为：
        >$$y=f_{w}(x)$$

        很多时候，我们希望得到一个“稳健”的模型。何为稳健？一般来说有两种含义，一是对于参数扰动的稳定性，比如模型变成了$f_{w}+\Delta w(x)$后是否还能达到相近的效果？如果在动力学系统中，还要考虑模型最终是否能恢复到 fw(x)；二是对于输入扰动的稳定性，比如输入从 x 变成了 x+Δx 后，$f_{w}(x+\Delta x)$ 是否能给出相近的预测结果。
        所以，大多数时候我们都希望模型对输入扰动是不敏感的，这通常能提高模型的泛化性能。于是 Lipschitz 提出了一个更具体的约束，那就是存在某个常数 C（它只与参数有关，与输入无关），使得下式恒成立
        >$$\left\|f_{w}\left(x_{1}\right)-f_{w}\left(x_{2}\right)\right\| \leq C(w) \cdot\left\|x_{1}-x_{2}\right\|$$

        也就是说，希望整个模型被一个线性函数“控制”住。这便是 L 约束了。
        换言之，在这里我们认为满足 L 约束的模型才是一个好模型。并且对于具体的模型，我们希望估算出 C(w) 的表达式，并且希望 C(w) 越小越好，越小意味着它对输入扰动越不敏感，泛化性越好。

### 导数与微分
1. **导数**
    设函数y=f(x)在点x0的某个区域有定义，如果$\frac{\Delta y}{\Delta x}$在当Δx->0时极限存在，则称函数y=f(x)在x0可导，这个极限是函数y=f(x)在x0的导数：
    >$$f^{\prime}\left(x_{0}\right)=\lim _{\Delta x \rightarrow 0} \frac{\Delta y}{\Delta x}=\lim _{\Delta x \rightarrow 0} \frac{f\left(x_{0}+\Delta x\right)-f\left(x_{0}\right)}{\Delta x}$$

2. **求导法则与公式**
    * **求导法则**
        >$$[u(x) \pm v(x)]^{\prime}=u^{\prime}(x) \pm v^{\prime}(x)$$

        >$$[u(x) v(x)]^{\prime}=u^{\prime}(x) v(x)+u(x) v^{\prime}(x)$$

        >$$\left[\frac{u(x)}{v(x)}\right]^{\prime}=\frac{u^{\prime}(x) v(x)-u(x) v^{\prime}(x)}{v^{2}(x)}(v(x) \neq 0)$$

    * **反函数求导法则**
        >$$\left[f^{-1}(x)\right]^{\prime}=\frac{1}{f^{\prime}(y)} 或 \quad \frac{d y}{d x}=\frac{1}{\frac{d x}{d y}}$$

    * **复合函数求导法则**
        存在$y=f(u)$在点$u=g(x)$处可导，则导数为
        >$$\frac{\mathrm{d} y}{\mathrm{d} x}=f^{\prime}(u) \cdot g^{\prime}(x) 或 \quad \frac{\mathrm{d} y}{\mathrm{d} x}=\frac{\mathrm{d} y}{\mathrm{d} u} \cdot \frac{\mathrm{d} u}{\mathrm{d} x}$$

    * **隐函数导数**
        函数 $y=f(x)$ 表示显函数
        方程 $F(x,y)=0$ 所确定的y是x的函数称为隐函数
        **一般方法**：隐函数左右两边同时对x求导
        **幂指函数**：对于一般幂指函数 $y = a ^ { x } ( a > 0 ) $，可以采取两边取对数，然后求导
        >$\ln y = x \ln a \ => \ (\ln y)^{\prime} = ( x \ln a)^{\prime}  $

    * **导数公式**
        >$( C ) ^ { \prime } = 0 \ \ \ \ \  ( x ^ { \mu } ) ^ { \prime } = \mu x ^ { \mu - 1 }  $
        $ ( \sin x ) ^ { \prime } = \cos x \ \ \ \ \ ( \cos x ) ^ { \prime } = - \sin x  $
        $( \tan x ) ^ { \prime } = \sec^ { 2 } x \ \ \ \ \  ( \cot x ) ^ { \prime } = - \cos ^ { 2 } x  $
        $( \sec x ) ^ { \prime } = \sec x\tan x \ \ \ \ \ ( \csc x ) ^ { \prime } = - \cos x \cot x  $
        $( a ^ { x } ) ^ { \prime } = a ^ { x } \ln a \ \ \ \ \ ( e ^ { x } ) ^ { \prime } = e ^ { x }  $
        $( \log _ { a } x ) ^ { \prime } = \frac { 1 } { x \ln a } \ \ \ \ \ ( \ln x ) ^ { \prime } = \frac { 1 } { x }  $
        $( \arcsin x ) ^ { \prime } = \frac { 1 } { \sqrt { 1 - x ^ { 2 } } } \ \ \ \ \ ( \arccos x ) ^ { \prime } = - \frac { 1 } { \sqrt { 1 - x ^ { 2 } } }  $
        $( \arctan x ) ^ { \prime } = \frac { 1 } { 1 + x ^ { 2 } } \ \ \ \ \ ( \arctan x ) ^ { \prime } = - \frac { 1 } { 1 + x ^ { 2 } }  $
        
    * **莱布尼茨公式**
        莱布尼茨公式用于对两个函数的乘积求取其高阶导数
        一般的，如果函数u=u(x)与函数v=v(x)在点x处都具有n阶导数，那么此时有
        >$$(uv) ^ {(n)} = \sum\limits _ { k = 0 } ^ { n } C _ { n } ^ { k } u ^ { ( n - k ) } v ^ {(k)}$$
        
        注意区分二项式展开公式
        >$$( u + v ) ^ { n } = \sum\limits _ { k = 0 } ^ { n } C _ { n } ^ { k } u ^ { n - k } v ^ { k }$$

3. **微分**
    设函数y=f(x)在某个领域内有定义，$x_{0}$及$x_{0}+\Delta x$在这区间内，如果增量：
    >$$\Delta y=f\left(x_{0}+x\right)-f\left(x_{0}\right)$$

    可表示为
    >$$\Delta y=A \Delta x+o(\Delta x)$$

    其中A是不依赖$\Delta x$的常数，$o(\Delta x)$是指$\Delta x$趋于0时的高阶无穷小，那么称函数y=f(x)在点$x_{0}$是可微的，而$A\Delta x$叫做函数在点$x_{0}$相应于自变量增量$\Delta x$的微分，记作$\mathrm{d} y$
    >$$\mathrm{d} y=A \Delta x$$
    
4. **微分与导数的关系**
    如果函数f(x)在点$x_{0}$可微，则f(x)在点$x_{0}$也一定可导，且$A=f^{\prime}\left(x_{0}\right)$。反之，如果f(x)可导，存在下式：
    >$$\lim _{\Delta x \rightarrow 0} \frac{\Delta y}{\Delta x}=f^{\prime}\left(x_{0}\right)$$

    根据极限与无穷小的关系转化上式，当$\Delta x \rightarrow 0$时
    >$$\frac{\Delta y}{\Delta x}=f^{\prime}\left(x_{0}\right)+\alpha$$

    其中当$\Delta x \rightarrow 0$时，$\alpha$无穷小，所以
    >$$\Delta y=f^{\prime}\left(x_{0}\right) \Delta x+o(\Delta x)$$

    上述公式表示增量和微分的关系

    函数f(x)在点$x_{0}$可微的充分必要条件是函数f(x)在点$x_{0}$可导
    >$$\mathrm{d} y=f^{\prime}\left(x_{0}\right) \Delta x$$

    函数的微分表达式
    >$$\mathrm{d} y = f ^ { \prime } ( x ) \mathrm{d} x  $$

6. **微分的应用**
    * **近似计算**
        >$$f ( x ) \approx f ( x _ { 0 } ) + f ^ { \prime } ( x _ { 0 } ) ( x - x _ { 0 } )  $$

### 微分中值定理
1. **罗尔中值定理**
    如果函数$y=f(x)$满足下列条件：
    1) 在闭区间[a,b]上连续
    2) 在开区间(a,b)上可导
    3) f(a) = f(b)

    则开区间(a,b)至少存在一点$\xi$，使得
    >$$f^{\prime}(\xi)=0$$

2. **拉格朗日中值定理**
    如果函数$y=f(x)$满足下列条件：
    1) 在闭区间[a,b]上连续
    2) 在开区间(a,b)上可导

    则开区间(a,b)至少存在一点$\xi$，使得
    >$$f(b)-f(a)=f^{\prime}(\xi)(b-a)$$

    存在$x_{0}, x_{0} + \Delta x\in (a,b)$，则有
    >$$f ( x _ { 0 } + \Delta x ) - f ( x _ { 0 } ) = f ^ { \prime } ( x _ { 0 } + \theta \Delta x ) \cdot \Delta x \ \ \ \ ( 0 < \theta < 1 )  $$
    
   此公式又称有限增量公式或有限增量定理，它与微分公式的近似有区别
   >$$f ( x + \Delta x ) \approx f ( x ) + f ^ { \prime } ( x ) \Delta x \ \ \ (微分，近似)$$

   >$$f ( x + \Delta x ) = f ( x ) + f ^ { \prime } ( x + \theta A x ) \Delta x \ \ \ (拉氏，精确)$$

3. **柯西中值定理**
    如果函数$f(x)$和$g(x)$满足
    1) 在闭区间[a,b]上连续
    2) 在开区间(a,b)上可导
    3) 对任意$x \in ( a , b )  $，$g ^ { \prime } ( x ) \neq 0  $

    则存在
    >$$\frac { f ( b ) - f ( a ) } { g ( b ) - g ( a ) } = \frac { f ^ { \prime } ( \xi ) } { g^{\prime} ( \xi ) }  $$

### 导数的应用
1. **洛必达法则**
    用来求极限的方法，在计算$0/0$ 或 $\infty/\infty $的极限时，可用下式：
    >$$\lim\limits _ { x \rightarrow a } \frac { f ( x ) } { F ( x ) } = \lim\limits _ { x \rightarrow a } \frac { f ^ { \prime } ( x ) } { F ^ { \prime } ( x ) }  $$

2. **泰勒公式**
    泰勒公式是将一个在 $x=x0$ 处具有 $n$ 阶导数的函数 $f(x)$ 利用关于 $(x-x0)$ 的 $n$ 次多项式来逼近函数的方法([通俗理解泰勒公式](https://www.zhihu.com/question/21149770))
    函数 $f(x)$ 在包含 $x_{0}$ 的某个闭区间 $[a,b]$ 上具有 $n$ 阶导数，且在开区间 $(a,b)$ 上具有 n+1 阶导数，则对闭区间上任意一点 $x$ 成立： 
    >$$f ( x ) = \frac { f ( x _ { 0 } ) } { 0 ! } + \frac { f ^ { \prime } ( x _ { 0 } ) } { 1 ! } ( x - x _ { 0 } ) + \frac { f ^ { \prime \prime } ( x _ { 0 } ) } { 2 ! } ( x - x _ { 0 } ) ^ { 2 } + \cdots + \frac { f ^ { (n) } ( x _ { 0 } ) } { n ! } ( x - x _ { 0 } ) ^ { n } + R _ { n } ( x )  $$

    >$$f ( x ) = \sum\limits _ { n = 0 } ^ { N } \frac { f ^ { ( n ) } ( x_{0} ) } { n ! } ( x - x_{0} ) ^ { n } + R _ { n } ( x )  $$

    其中多项式部分称为函数在 $x_{0}$ 处的泰勒展开式。$R _ { n } ( x )  $ 是泰勒公式的余项且是 $( x - x_{0} ) ^ { n }  $ 的高阶无穷小
    如果$x_{0}=0$, 就是麦克劳伦公式
    >$$f ( x ) = \sum\limits _ { n = 0 } ^ { N } \frac { f ^ { \prime } ( 0 ) } { n ! } x ^ { n } + R _ { n } ( x )  $$

    泰勒公式的余项代数式如下，其中$x_{0} < \theta < x  $
    >$$R _ { n } ( x ) = \frac { f ^ { ( n + 1 ) } ( \theta ) } { ( n + 1 ) ! } ( x - x_{0} ) ^ { ( n + 1 ) }  $$

    当 $n=0$ 时，根据泰勒公式得到
    >$$f ( x ) = f ( x_{0} ) + f ^ { \prime } ( \theta ) ( x - x_{0} )  $$

    所以拉格朗日中值定理根本就是 $n=0$ 时的泰勒公式

### 不定积分及其计算
1. **概念与性质**
    在区间 $I$ 上定义两个函数，且满足 $F^{\prime}(x)=f(x)$ 或 $dF(x)=f(x)dx$，则称 $F(x)$ 为 ${f(x)}$ 在 ${I}$ 区间上的原函数。表示为
    >$$\int f(x)dx=F(x)+C$$

    积分的性质
    >$$\int k f(x) d x = k \int f( x ) d x \quad ( k \neq 0 ) $$
    
    >$$\int \lbrack f ( x ) \pm g ( x ) \rbrack d x = \int f ( x ) d x \pm \int g ( x ) d x  $$
    
    >$$\lbrack \int f ( x ) d x + \int g ( x ) d x \rbrack^{\prime} = \lbrack \int f ( x ) d x \rbrack ^{\prime} + \lbrack \int g ( x ) d x \rbrack ^{\prime}$$

2. **基本积分表**
    >$\int k dx = k x + C $
    $\int x ^ { u } d x = \frac { x ^ { u + 1 } } { u + 1 } + C  $
    $\int \frac { 1 } { x } d x = \ln | x | + C  $
    $\int \frac { 1 } { 1 + x ^ { 2 } } d x = \arctan x + C = - a r c \cot x + C  $
    $\int \frac { 1 } { \sqrt { 1 - x ^ { 2 } } } = \arcsin x + C = - \arccos x + C  $
    $\int \cos x d x = \sin x + C  $
    $\int \sin x d x = - \cos x + C  $
    $\int \frac{1}{\cos^{2}x}dx = \int \sec^{2}xdx = \tan x + C  $
    $\int \frac{1}{\sin^{2}x}dx = \int \cos^{2}xdx = - \cot x + C  $
    $\int \sec x \tan x dx = \sec x + C$
    $\int e^{x} dx = e^{x} + C$
    $\int a^{x} dx = \frac {a^{x}} {\ln a} + C$

3. **计算方法**
    * **换元积分法**
        设 $f(u)$ 具有原函数 $F(u)$，即 $F^{ \prime }( u )=f(u)$，$\int f ( u ) d u = F ( u ) + C  $
        如果 $u = \varphi(x)$，且设 $\varphi(x)$ 可微，根据复合函数微分法
        >$$d F \lbrack \varphi ( x ) \rbrack = f \lbrack \varphi ( x ) \rbrack \varphi ^ { \prime } ( x ) d x  $$
    
        其不定积分为
        >$$\int f \lbrack \varphi ( x ) \rbrack \varphi ^ { \prime } ( x ) d x  = F \lbrack \varphi ( x ) \rbrack + C = \lbrack \int f ( u ) d u \rbrack _ { u = \varphi (x) }   $$
    
    * **分部积分法**
        两个函数的乘积导数为

        >$$( u v ) ^ { \prime } = u ^ { \prime } v + u v ^ { \prime }    =>  uv^{\prime}=(uv)^{\prime}-u^{\prime}v$$

        两边进行不定积分
        >$$\int uv^{ \prime } d x = uv - \int u^{\prime} vdx$$

        >$$\int udv = uv - \int vdu$$


### **定积分的计算及其应用**
1. **概念**
    函数 $f(x)$ 在区间 $[a,b]$ 上有界，在区间 $[a,b]$ 内插入 $n-1$ 个点: $a=x_{0}<x_{1}<x_{2}<...<x_{n-1} < x_{n} = b$，记 $\Delta x=x_{i}-x_{i-1}(i=1,\cdots,n)$，$\lambda = \max \lbrace \Delta x _ { 1 } , \Delta x _ { 2 } , \cdots , \Delta x _ { n } \rbrace$，在小区间 $[x_{i-1},x_{i}]$上任取一点，$\xi _ { i } ( i = 1 , 2 , \cdots , n )$，做和 $\sum\limits _ { i = 1 } ^ { n } f ( \xi _ { i } ) \Delta x _ { i } $；如果当 $\lambda \rightarrow 0  $时，上述的和存在极限，且极限值与区间[a,b]的分法和 $\xi_{i}$ 的取法无关，则称此极限为函数 $f(x)$的定积分：
    >$$\int _ { a } ^ { b } f ( x ) d x = \lim\limits _ { \lambda \rightarrow 0 }\sum\limits_{i=1}^{n}f(\xi_{i}) \Delta x_{i}$$
    
    此时称 $f(x)$ 在 $[a,b]$ 上可积
    注意：定积分表示一个数，而不定积分是一个函数族，分别对定积分和不定积分求导
    >$$\frac { d } { d x } \int f ( x ) d x = f ( x ) \ \ \ \ \ \ \quad \frac { d } { d x } \int _ { a } ^ { b } f ( x ) d x = 0  $$

2. **性质**
    >$$\int _ { a } ^ { b } \lbrack f ( x ) \pm g ( x ) \rbrack d x = \int _ { a } ^ { b } f ( x ) d x \pm \int _ { a } ^ { b } g ( x ) d x $$

    >$$\int _ { a } ^ { b } k f ( x ) d x = k \int _ { a } ^ { b } f ( x ) d x $$

    >$$\int _ { a } ^ { b } f ( x ) d x = \int _ { a } ^ { c } f ( x ) d x + \int _ { e } ^ { b } f ( x ) d x \ \ (a < c < b)$$
 
    设M和m分别是f(x)在区间[a,b]上的最大值和最小值，则
    >$$m ( b - a ) \leq \int _ { a } ^ { b } f ( x ) d x \leq M ( b - a )  $$

3. **定积分中值定理**
    若f(x)在闭区间上[a,b]连续,则积分区间[a,b]上至少存在一个点$\xi$，使
    >$$\int _ { a } ^ { b } f ( x ) d x = f ( \xi ) ( b - a ) \ \ \quad ( a \leq \xi \leq b )  $$

4. **牛顿-莱布尼兹公式**
    如果函数 $F(x)$ 是连续函数 $f(x)$ 在区间 $[a,b]$ 上的一个原函数
    >$$\int _ { a } ^ { b } f (x) dx= F(b)-F(a)$$

    揭示了定积分与被积函数的原函数或不定积分之间的关系

5. **反常积分(广义积分)**
    * **无穷限的反常积分**
        设函数 $f(x)$ 在区间 $\lbrack a, +\infty)$ 上连续，取 $t>a$，其广义积分为
        >$$\int _ { a } ^ { +\infty } f ( x ) d x = \lim\limits _ { t \rightarrow + \infty } \int _ { a } ^ { t } f ( x ) d x  $$

        如果极限 $\lim\limits _ { t \rightarrow + \infty } \int _ { a } ^ { t } f ( x ) d x$ 存在，则称广义积分收敛，如果不存在，则称广义积分发散

        类似在区间 $(-\infty, b \rbrack$ 上连续，其广义积分为
        >$$\int _ { -\infty } ^ { b } f ( x ) d x = \lim\limits _ { t \rightarrow - \infty } \int _ { t } ^ { b } f ( x ) d x  $$

        同理，如果极限 $\lim\limits _ { t \rightarrow - \infty } \int _ { t } ^ { b } f ( x ) d x  $ 存在，则称广义积分收敛，如果不存在，则称广义积分发散

        如果函数在区间 $(-\infty, +\infty)$ 上连续，其广义积分为
        >$$\int _ { - \infty } ^ { \infty } f ( x ) d x = \int _ { - \infty } ^ { o } f ( x ) d x + \int _ { 0 } ^ { \infty } f ( x ) d x = \lim\limits _ { a \rightarrow - \infty } \int _ { a } ^ { 0 } f ( x ) d x + \lim\limits _ { b \rightarrow \infty } \int _ { 0 } ^ { b } f ( x ) d x  $$

        同理，如果极限存在，则称广义积分收敛，如果不存在，则称广义积分发散

    * **无界函数的反常积分**
        如果函数在点 $a$ 的任一邻域内都无界，那么点 $a$ 称 $f(x)$ 函数的瑕点（也称无界间断点）
        设 $f(x)$ 在 $(a,b]$ 上连续，则其反常积分如下。如果极限存在，则称反常积分收敛。不存在，则称反常积分发散
        >$$\int _ { a } ^ { b } f ( x ) d x = \lim\limits _ { t \rightarrow a } \int _ { t } ^ { b } f ( x ) d x  $$

        计算无界函数的反常积分，可用牛顿-莱布尼兹公式。那么 $f(x)$ 在 $(a,b]$ 上的反常积分为
        >$$\int _ { a } ^ { b } f ( x ) d x = F ( b ) - \lim\limits _ { x \rightarrow a^{+} } F ( x ) = F (b)-F (a^{+})$$

        如果极限 $\lim\limits _ { x \rightarrow a^{+} } F ( x )$ 存在，则称反常积分收敛，否则发散

    * **反常积分的审敛法**
        我们通常对反常积分是发散还是收敛很感兴趣，然而计算极限往往令人沮丧，幸而我们了解增长和衰减速率，将被积函数替换成更快或更慢的函数，以此判断反常积分的收敛性，这种方法就是审敛法。

        * **无穷限反常积分审敛法**
            * **比较审敛法1**
                设函数 $f(x)$ 在区间 $[a, +\infty)$ $(a > 0)$ 上连续，且 $f ( x ) \geq 0$

                如果存在常数 $M > 0$ 及 $p > 1$，使得 $f ( x ) \leq \frac { M } { x ^ { p } } , ( a \leq x < + \infty ) $，则 $\int _ { a } ^ { + \infty } f ( x ) d x$ 收敛

                如果存在常数 $N>0$，使得 $f ( x ) \geq \frac { N } { x } ( a \leq x < + \infty ) $，则 $\int _ { a } ^ { + \infty } f ( x ) d x$ 发散

            * **极限审敛法1**
                设函数 $f(x)$ 在区间 $[a, +\infty)$ $(a > 0)$ 上连续，且 $f ( x ) \geq 0$

                如果存在常数 $p > 1$，使得 $\lim\limits _ { x \rightarrow \infty } x ^ { p } f ( x )$ 存在，则 $\int _ { a } ^ { + \infty } f ( x ) d x$ 收敛

                如果 $\lim\limits _ { x \rightarrow + \infty } x f ( x ) = d > 0 $ (或 $\lim\limits _ { x \rightarrow + \infty } x f ( x ) = + \infty $)，则 $\int _ { a } ^ { + \infty } f ( x ) d x $ 发散
        
        * **无界函数的反常积分审敛法**
            * **比较审敛法2**
                设函数 $f(x)$ 在区间 $(a, b]$ 上连续，且 $f ( x ) \geq 0$，$x$ 是瑕点
                
                如果存在常数 $M > 0$ 及 $q<1$，使得 $f ( x ) \leq \frac { M } { ( x - a ) ^ { q } } ( a < x \leq b )  $，则称反常积分 $\int _ { a } ^ { b } f ( x ) d x $ 收敛

                如果存在常数 $N > 0$ ，使得 $f ( x ) \geq \frac { N } { x - a } \quad ( a < x \leq b )$，则称反常积分 $\int _ { a } ^ { b } f ( x ) d x $ 发散

            * **极限审敛法2**
                设函数 $f(x)$ 在区间 $(a, b]$ 上连续，且 $f ( x ) \geq 0$，$x$ 是瑕点
                
                如果存在常数 $ 0 < q < 1 $，使得 $\lim\limits _ { x \rightarrow a ^ { + } } ( x - a ) ^ { q } f ( x ) $ 存在，则称反常积分 $\int _ { a } ^ { b } f ( x ) d x $ 收敛

                如果 $\lim\limits _ { x \rightarrow a ^ { + } } ( x - a ) f ( x ) = d > 0 $ （或 $\lim\limits _ { x \rightarrow a ^ { + } } ( x - a ) f ( x ) = + \infty $），则称反常积分 $\int _ { a } ^ { b } f ( x ) d x $ 发散

6. **Gamma函数**
    * **函数定义**
        （此函数广义积分的收敛性分析此处略）
        >$$\Gamma ( s ) = \int _ { 0 } ^ { + \infty } e ^ { -x }x^{s-1} d x \quad ( s > 0 ) $$

        函数特点：
        （1）$s-1<0$ 时 $x=0$ 是被积函数的瑕点
        （2）广义积分 $\int _ { 0 } ^ { + \infty } e ^ { -x }x^{s-1} d x $ 对 $s>0$ 均收敛
        $\Gamma $ 函数如图
        ![](https://cn.bing.com/th?id=OIP.DCp106myuC-kyqC3WAVlmwAAAA&pid=Api&rs=1)

    * **函数性质**
        * 递推公式 $\Gamma (s+1)=s\Gamma (s)(s>0)$
            >$\Gamma ( 2 ) = 1 \cdot \Gamma ( 1 ) = 1$
            $\Gamma ( 3 ) = 2 \cdot \Gamma ( 2 ) = 2!$
            $\Gamma ( 4 ) = 3 \cdot \Gamma ( 3 ) = 3!$
            ...
            $\Gamma(n+1) = n!$ 
            把 $\Gamma$ 函数可看成阶乘的推广

        * 当 $s \rightarrow 0^{+}$ 时，$\Gamma(s) \rightarrow +\infty$

        * $\Gamma ( s ) \Gamma ( 1 - s ) = \frac { \pi } { \sin \pi s } \ \ \ ( 0 < s < 1 )$。此公式又称“余元公式”
            当 $s = \frac { 1 } { 2 }$ 时，
            >$\Gamma ( \frac { 1 } { 2 } ) = \sqrt { \pi }$

        * 将 $\Gamma$ 函数进行变换，$x=u^{2}$，则 $\Gamma ( s ) = 2 \int _ { 0 } ^ { \infty } e ^ { - u ^ { 2 } } u ^ { 2 s - 1 } du $，再令 $t=2s-1$，既有 $\int _ { 0 } ^ { \infty } e ^ { - u ^ { 2 } } u ^ { t } d u = \frac { 1 } { 2 } \Gamma ( \frac { 1 + t } { 2 } ) , t > -1$
            令 $s = \frac { 1 } { 2 }$，可得泊松积分
            >$\int _ { - \infty } ^ { \infty } e ^ { - x ^ { 2 } } d x = \sqrt { \pi } $
            $\int _ { 0 } ^ { \infty } e ^ { - x ^ { 2 } } d x = \frac { \sqrt { \pi } } { 2 }  $


### **微分方程（常微分方程）**

* **相关概念**
    含有自变量x、函数y及函数各阶导数的方程称为微分方程，其中函数y是一元函数，导数项必须有，自变量x或函数y可以没有
    >$$ F ( x , y , y ^ { \prime } , \cdots , y ^ { ( n ) } ) = 0 $$

    微分方程中导数的最高项称为微分方程的阶
    满足微分方程函数的解称为微分方程的解，含有独立任意常数与微分方程的阶数相同的解，称为通解
    \
    线性相关与线性不相关
	设 $y_{1}(x),y_{2}(x) ... y_{n}(x)$ 为定义在区间 $I$ 上的 $n$ 个函数，如果存在n个不全为零的的常数 $k_{1},k_{2} ... k_{n}$，使得下式成立，则称这 $n$ 个函数在区间 $I$ 上线性相关，否则线性不相关
    >$$k _ { 1 } y _ { 1 } + k _ { 2 } y _ { 2 } + \cdots + k _ { n } y _ { n } = 0 $$


1. **一阶微分方程的初等解法**
    >$$F ( x , y , y ^ { \prime } ) = 0$$

    * **可分离变量的微分方程**
        如果一个一阶微分方程可写成
        >$$g ( y ) d y = f ( x ) d x $$

        两边积分
        >$$\int g ( y ) d y = \int f ( x ) d x  $$

        $G(y)$ 和 $F(x)$ 分别是 $g(y)$ 和 $f(x)$ 的原函数
        >$$G(y) = F(x) + C$$

    * **齐次方程**
        如果一阶方程可写成如下，那么就称为齐次方程
        >$$\frac { d y } { d x } = \varphi ( \frac { y } { x } )  $$

        对于齐次方程，令 $u = \frac { y } { x }$，$y=xu$，所以$\frac { d y } { d x } = u + x \frac { d u } { d x }  $，代入原方程
        >$$x \frac { d u } { d x } = f ( u ) - u  $$

        分离变量
        >$$\frac {du} {f(u)-u} = \frac {dx}{x}$$

        上述是可分离变的微分方程，求出解后，用 $u= \frac {y}{x}$ 代入即得到通解

    * **一阶线性微分方程**
        标准形式
        >$$\frac { d y } { d x } + P ( x ) y = Q ( x ) $$
            
        >当 $Q(x) = 0$ 时，称为齐次线性方程
        >当 $Q(x) \neq 0$ 时，称为非齐次线性方程

        **一阶齐次线性微分方程通解**
        >$$y = Ce ^ {-\int P (x) dx} \ \ (C = \pm e ^ { c_{1}}) $$

        **一阶非齐次线性微分方程通解**
        >$$y = C e ^ { - \int P ( x ) d x } + e ^ { - \int P ( x ) d x } \int Q ( x ) e ^ { \int P(x)dx }dx $$

        **一阶非齐次线性微分方程的通解 = 齐次方程的通解 + 非齐次方程的一个特解**

        推导过程
        （1）齐次方程
        >分离变量得
        >$\frac { d y } { y } = - P ( x ) d x$
        >两边积分得到通解
        >$\ln |y| = -\int P (x) dx + C_{1}$
        >$y = Ce ^ {-\int P (x) dx} \ \ (C = \pm e ^ { c_{1}}) $

        （2）非齐次方程
        >上述使用**常数变易法**进行求解，此方法把齐次通解中的 $C$ 换成 $x$ 的未知函数 $u(x)$
        >$y = u ( x ) \cdot e ^ { - \int P ( x ) d x } (u(x) \neq 0)$       (1)
        >对 $y$ 进行求导
        >$\frac { d y } { d x } = u ^ { \prime } e ^ {  - \int P(x) dx } - u P ( x ) e ^ { - \int P ( x ) d x}$
        >将 $y$ 和 $\frac {dy}{dx}$ 带入非齐次线性方程中 $\frac { d y } { d x } + P ( x ) y = Q ( x ) $
        >$\Rightarrow  u ^ { \prime } e ^ {- \int P ( x ) d x} - u P ( x ) e ^ { - \int P ( x ) d x } + P ( x ) u e ^ { - \int P(x)dx } = Q ( x )  $
        >$\Rightarrow u ^ { \prime } e ^ { - \int  P ( x ) d x }$
        >$\Rightarrow u^{\prime} = Q(x)e^{\int P(x)dx}$
        >两边积分
        >$\int u^{\prime} du = \int Q ( x ) e ^ { \int P ( x ) d x }dx  $
        >$u = \int Q ( x ) e ^ { \int P ( x ) d x }dx + C $
        >将 $u$ 带入上式(1)
        >$y = C e ^ { - \int P ( x ) d x } + e ^ {-\int P ( x ) d x } \int Q ( x ) e ^ { \int P(x)dx }dx $

    * **伯努利方程**
        一般形式
        >$$\frac {dy}{dx} + P(x)y = Q(x)y^{n} \ \ \ (n \neq 0,1)$$

        当 $n=0,1$ 时，这是线性方程。当 $n \neq 0,1$ 时，这是非线性方程
        上述方程进行变换
        >$y ^ { - n } \frac { d y } { d x } + P ( x ) y ^ { 1 - n } = Q ( x )  $

        >$z=y^{1-n}$

        >$\frac { d z } { d x } = ( 1 - n ) y ^ { - n } \frac { d y } { d x }  $

        最终得到线性方程
        >$\frac { d z } { d x } + ( 1 - n ) P ( x ) z = ( 1 - n ) Q ( x )  $

3. **高阶线性微分方程**
	* **二阶线性微分方程**
        齐次方程
        >$$y ^ { \prime \prime } + P ( x ) y ^ { \prime } + Q ( x ) y = 0$$
 
        如果 $y_{1}(x)$ 和 $y_{2}(x)$ 是两个线性无关的解（$\frac {y_{1}}{y_{2}} \neq C$），那么通解为
        >$$y = C_{1} y_{1} ( x ) + C _ { 2 } y _ { 2 } ( x ) \ \ \ \ (C_{1}, C_{2}是任意常数) $$
        
        非齐次方程
        >$$y ^ { \prime \prime } + P ( x ) y ^ { \prime } + Q ( x ) y = f(x)$$
        
        设 $y^{*}(x)$ 是二阶非齐次方程的一个特解，$Y(x)$ 是上述齐次方程的通解，那么非齐次方程的通解为
        >$$y = Y(x) + y^{*}(x)$$

	* **高阶线性微分方程**
        齐次方程
    	>$$y ^ { (n) } + a _ { 1 } ( x ) y ^ { (n-1) } + \cdots + a _ { n-1 } ( x ) y ^ { \prime } + a _ { n } ( x ) y = 0  $$

        如果 $y_{1}(x), y_{2}(x) \cdots  y_{n}(x)$ 是n阶齐次方程的n个线性无关的解，那么通解为
        >$$y = C_{1} y_{1} ( x ) + C _ { 2 } y _ { 2 } ( x ) + \cdots + C _ { n } y _ { n } ( x ) \ \ \ \ (C_{1}, C_{2} \cdots C_{n} 是常数) $$

        非齐次方程
        >$$y ^ { n } + a _ { 1 } ( x ) y ^ { n - 1 } + \cdots + a _ { n - 1 } ( x ) y ^ { \prime } + a _ { n } ( x ) y = f ( x )  $$

4. **常系数线性微分方程**
    * **常系数齐次线性微分方程**
        * **二阶常系数齐次微分方程**
            >$$y ^ { \prime \prime } + p y ^ { \prime } + q y = 0 \ \ \ (p, q 常数) $$

            设 $y=e^{rx}$，$r$ 为常数，那么
            >$$y^{\prime} = re^{rx} \ \ \ y^{\prime \prime} = r^{2}e^{rx} $$

            代入微分方程
            >$$(r ^ { 2 } + p r + q)e^{rx} = 0  $$
            
            因为 $e^{rx} \neq 0$，所以
            >$$r ^ { 2 } + p r + q = 0$$

            上式叫做微分方程的**特征方程**，其特征方程的两个根为
            >$$r _ { 1,2 } = \frac { - p \pm \sqrt { p ^ { 2 } - 4 q } } { 2 }  $$

            **通解**
            (1) 两个不相等的实根 $r_{1}, r_{2}$，通解为 $y = C_{1}e^{r_{1}x} + C_{2}e^{r_{2}x}$
            (2) 两个相等的实根 $r_{1} = r_{2}$，通解为 $y = (C_{1}+C_{2}x)e^{r_{1}x}$
            (3) 一对共轭复根 $r_{1,2} = \alpha \pm i \beta $，通解为 $y = e^{\alpha x}(C_{1}\cos \beta x + C_{2}\sin \beta x)$

        * **n阶常系数齐次微分方程**
            >$$y ^ { ( n ) } + p _ { 1 } y ^ { ( n - 1 ) } + \cdots + p _ { n - 1 } y ^ { \prime } + p _ { n } y = f ( x ) \ \ \ \ (p_{1} \cdots p_{n} 都是常数) $$

            用记号 $D$（微分算子）表示对 $x$ 求导的运算 $\frac {d} {dx}$，把 $\frac {dy}{dx}$ 记作 $Dy$，$\frac {d^{n}y}{dx^{n}}$ 记作 $D^{n}y$。
            上述方程可以写作
            >$$( D ^ { n } + p _ { 1 } D ^ { n - 1 } + \cdots + p _ { n - 1 } D + p _ { n } ) y = 0  $$

            >$$L ( D ) = D ^ { n } + p _ { 1 } D ^ { n - 1 } + \cdots + p _ { n - 1 } D + p _ { n }  $$

            L(D) 叫做微分算子 D 的 n 次多项式，可记作
            >$$L(D)y = 0$$

            令 $y=e^{rx}$，由于 $De^{rx} = re^{rx} \dots D^{n}e^{rx} = r^{n} e^{rx}$，所以 $L(D)e^{rx}=L(r)e^{rx}$ 代入上式
            >$$L(r)e^{rx} = 0 \ \Rightarrow \ L(r) = 0$$

            **特征方程**
            >$$r ^ { n } + p _ { 1 } r ^ { n - 1 } + p _ { 2 } r ^ { n - 2 } + \cdots + p _ { n - 1 } r + p_{ n } = 0  $$

            **通解**
            (1) 单实根 $r$，给出一项 $Ce^{rx}$
            (2) 一对单负根 $r_{ 1,2 } = \alpha \pm i \beta$，给出两项 $e^{\alpha x}(C_{1} \cos \beta x + C_{2} \sin \beta x)$
            (3) $k$ 个实根 $r$，给出 $k$ 项 $e ^ { r x} ( C _ { 1 } + C _ { 2 } x + \cdots + C _ { k } x ^ { k - 1 } )  $
            (4) 一对$k$重复根 $k_{1,2} = \alpha \pm i \beta$，给出 $2k$ 项 $e ^ { \alpha r } \lbrack ( C _ { 1 } + c _ { 2 } x + \cdots + C _ { k } x ^ { k - 1 } ) \cos \beta x + (D_{1}+D_{2}x+\dots+D_{k}x^{k-1}) \sin \beta x]  $
            另外，n次代数方程有n个根，每一项各含任意常数，n阶常系数齐次线性微分方程的通解为
            >$$y = C_{1} y _ { 1 } + C _ { 2 } y _ { 2 } + \cdots + C _ { n } y _ { n } $$

            例如：
            $y^{(4)} - 2 y^{\prime \prime \prime } + 5y^{\prime \prime }$ 特征方程 $r ^ { 4 } - 2 r ^ { 3 } + 5 r ^ { 2 } = 0  $ 的根为 $r_{1}=r_{2}=0$ 和 $r_{3,4}=1 \pm 2i$ 通解为
            >$y = C _ { 1 } + C _ { 2 } x + e ^ { x } ( C _ { 3 } \cos 2 x + C _ { 4 } \sin 2 x )  $
            
    * **常系数非齐次线性微分方程**
        略

5. **欧拉方程**
    变系数的线性常微分方程，一般来说都是不容易求解的。但是有些特殊的变系数线性常微分方程，则可以通过变量代换化为常系数线性微分方程
    >$$f ( x ) = x ^ { n } y ^ { ( n ) } + p _ { 1 } x ^ { n - 1 } y ^ { ( n - 1 ) } + \cdots + p _ { n - 1 } x y ^ { \prime } + p _ { n } y  $$

    推导过程
    >做变换 $x=e^{t}$ 或 $t=\ln x$
    >$\frac { d y } { d x } = \frac { d y } { d t } \cdot \frac { d t } { d x } = \frac { 1 } { x } \frac { d y } { d t }  $
    >$\frac { d ^ { 2 } y } { d x ^ { 2 } } = \frac { 1 } { x ^ { 2 } } ( \frac { d ^ { 2 } y } { d t ^ { 2 } } - \frac { d y } { d t } )  $
    >$\frac { d ^ { 3 } y } { d x ^ { 3 } } = \frac { 1 } { x ^ { 3 } } ( \frac { d ^ { 3 } y } { d t ^ { 3 } } - 3 \frac { d ^ { 2 } y } { d t ^ { 2 } } + 2 \frac { d y } { d t } )  $
    >$\frac { d ^ { 4 } y } { d x ^ { 4 } } = \frac { 1 } { x ^ { 4 } } ( \frac { d ^ { 4 } y } { d t ^ { 4 } } - 6 \frac { d ^ { 3 } y } { d t ^ { 3 } } + 1 1 \frac { d ^ { 2 } y } { d t ^ { 2 } } - 6 \frac { d y } { d t } ) $
    >如果采用记号D表示对t求导的运算
    >$x y ^ { \prime } = D y  $
    >$x ^ { 2 } y ^ { \prime \prime } = \frac { d ^ { 2 } y } { d t ^ { 2 } } - \frac { d y } { d t } = ( \frac { d ^ { 2 } } { d t ^ { 2 } } - \frac { d } { d t } ) y = ( D ^ { 2 } - D ) y = D ( D - 1 ) y  $
    >$x ^ { 3 } y ^ { m } = \frac { d ^ { 3 } y } { d t ^ { 3 } } - 3 \frac { d ^ { 2 } y } { d t ^ { 2 } } + 2 \frac { d y } { d t } = ( D ^ { 3 } - 3 D ^ { 2 } + 2 D ) y = D ( D - 1 ) ( D - 2 ) y $
    >一般有
    >$x ^ { k } y ^ { ( k ) } = D ( D - 1 ) \cdots ( D - k + 1 ) y  $
    >把它代入欧拉方程，便得到一个以 $t$ 为自变量的 $c$ 常系数线性微分方程

    例如求 $x ^ { 3 } y ^ {\prime \prime \prime} + x ^ { 2 } y ^ { \prime \prime } - 4 x y ^ { \prime } = 3 x ^ { 2 } $ 的通解
    >做变换 $x=e^{t}$ 或 $t=\ln x$
    >$D ( D - 1 ) ( D - 2 ) y + D ( D - 1 ) y - 4 D y = 3 e ^ { 2 t } $
    >$D ^ { 3 } y - 2 D ^ { 2 } y - 3 D y = 3 e ^ { 2 t }  $ （常系数非齐次线性微分方程）
    >$\frac { d ^ { 3 } y } { d t ^ { 3 } } - 2 \frac { d ^ { 2 } y } { d t ^ { 2 } } - 3 \frac { d y } { d t } = 3 e ^ { 2 t }   $
    >求解得
    >$y = c _ { 1 } + \frac { C _ { 2 } } { x } + C _ { 3 } x ^ { 3 } - \frac { 1 } { 2 } x ^ { 2 }  $


### 解析几何与向量代数

        >$$\overrightarrow { A B } = \overrightarrow { O B } - \overrightarrow { O A } = ( x _ { 2 } , y _ { 2 } , z _ { 2 } ) - ( x _ { 1 } , y _ { 1 } , z _ { 1 } ) = ( x _ { 2 } - x _ { 1 } , y _ { 2 } - y _ { 1 } , z _ { 2 } - z _ { 1 } )  $$

    * **方向角与方向余弦**
        非零向量 $\vec {r}$ 与三条坐标轴夹角 $\alpha$ $\beta$ $\gamma$ 称为向量 $r$ 的方向角
        设 $\overrightarrow { O M } = r = ( x , y , z )  $

3. **向量的乘法**
    * **数量积** 
        >$$a \cdot b = | a | | b | \cos \theta$$
        
        几何意义：向量a在向量b上投影的长度与向量b长度的乘积

        坐标系的形式，设 $a=a _ { x } i + a _ { y } j + a _ { z } k  $, $b=b _ { x } i + b _ { y } j + b _ { z } k  $
        >$$a \cdot b = a _ { x } b _ { x } + a _ { y } b _ { y } + a _ { z } b _ { z }  $$

    * **向量积（叉积、外积）**
        >$$\vec {c} = \vec {a} \times \vec {b}$$
        
        几何意义
        **大小**   $|\vec {c}| = |\vec {a}| |\vec {b}| \sin \theta $
        **方向**   $\vec {c}$ 的方向同时垂直于 $\vec {a}$ 和 $\vec {b}$，即垂直于 $\vec {a}$ 和 $\vec {b}$ 所决定的平面
        \
        设 $a = a _ { x } i + a_{y} j + a _ { z } k , b = b _ { z } i + b _ { y } j + b _ { z } k  $
        >$a \times b = (a _ { x } i + a_{y} j + a _ { z } k) \times (b _ { z } i + b _ { y } j + b _ { z } k)$
        >$= a _ { x } b _ { x } ( i \times i ) + a _ { x } b _ { y } ( i \times j ) + a _ { x } b _ { z } ( i \times k ) + $
        >$= a _ { y } b _ { x } ( j \times i ) + a _ { y } b _ { y } ( j \times j ) + a _ { y } b _ { z } ( j \times k ) + $
        >$= a _ { z } b _ { x } ( k \times i ) + a _ { z } b _ { y } ( k \times j ) + a _ { z } b _ { z } ( k \times k )  $

        单位向量的向量积 $i \times i = j \times j = k \times k = 0 , i \times j = k , j \times k = i, k \times i = j, j \times i = -k, k \times j = -i, i \times k = -j  $

        可写成        
        >$$a \times b= ( a _ { y } b _ { z } - a _ { z } b _ { y } ) i + ( a _ { z } b _ { x } - a _ { x } b _ { z } ) j + ( a _ { x } b _ { y } - a _ { y } b _ { x } ) k  $$

        可以用行列式表示
        >$$\left|\begin{array}{c} i & j & k \\ a _ { x } & a_{y} & a_{z} \\ b_{x} & b_{y} & b_{z} \end{array} \right|  $$

        >$$\left|\begin{array}{l} a _ { y } & a _ { z } \\ b_{y} & b_{z} \end{array} \right|\vec { i } - \left|\begin{array}{l} a _ { x } & a _ { z } \\ b_{x} & b_{z} \end{array} \right|\vec { j } + \left|\begin{array}{l} a _ { x } & a _ { y } \\ b_{x} & b_{y} \end{array} \right|\vec { k }$$

        **向量平行** $\vec {a} \times \vec {b} = 0$

        **向量垂直** $\vec {a} \cdot \vec {b} = 0$

    * **混合积**
        几何意义：平行六面体的体积
        >$$\lbrack \vec { a } \vec { b } \vec { c } \rbrack = ( \vec { a } \times \vec { b } ) \cdot \vec { c } = ( \vec { b } \times \vec { c } ) \cdot \vec { a } = ( \vec { c } \times \vec { a } ) \cdot \vec { b }  $$

### **多元函数微分**
1. **多元函数**
    * **概念**
        $n$ 为一个正整数，用 $R^{n}$ 表示 $n$ 元有序数组 $(x_{1}, x_{2} \cdots x_{n})$ 的全体所构成的集合，即
        >$$R ^ { n } = R \times R \times \cdots \times R = \lbrace ( x _ { 1 } , x _ { 2 } , \cdots , x _ { n } ) | x _ { i } \in R , i = 1 , 2 , \cdots , n \rbrace  $$

        $n$ 维空间上的点集 $D$，映射 $f:D \rightarrow R$ 就称为定义在 $D$ 上的 $n$ 元函数，记为
        >$$u = f ( x ) , x = ( x _ { 1 } , x _ { 2 } , \cdots , x _ { n } ) \in D  $$

    * **极限**
        略

    * **连续性**
        略

2. **偏导数**
    * **定义**
        设函数 $z=f(x,y)$ 在点 $(x_{0},y_{0})$ 的某一领域内有定义，当 $y$ 固定在 $y_{0}$, $x$ 固定在 $x_{0}$ 处有增量 $\Delta x$时，增量为 $ f ( x _ { 0 } + \Delta x , y _ { 0 } ) - f ( x _ { 0 } , y _ { 0 } )  $
        函数 $z=f(x,y)$ 在点 $(x_{0},y_{0})$ 处对 $x$ 的偏导数记为
        >$$\frac { \partial f } { \partial x } | _{x=x_{0}, y=y_{0}} \ \ \ 或 f_{x}(x_{0},y_{0})$$

        极限表示为
        >$$f _ { x } ( x _ { 0 } , y _ { 0 } ) = \lim\limits _ { \Delta x \rightarrow 0 } \frac { f ( x _ { 0 } + \Delta x , y _ { 0 } ) - f ( x _ { 0 } , y _ { 0 } ) } { \Delta x } $$

        几何意义
        >设 $M_{0}(x_{0},y_{0},f(x_{0},y_{0}))$ 为曲面 $z=f(x,y)$上的一个点，过 $M_{0}$ 作平面 $y=y_{0}$，截此曲面得一曲线，此曲线在平面 $y=y_{0}$ 上的曲线为 $z=f(x,y_{0})$。则偏导数 $f_{x}(x_{0},y_{0})$ 就是这曲线在点 $M_{0}$ 处的切线 $M_{0}T_{x}$ 对 $x$ 轴的斜率

    * **高阶偏导**
        函数 $z=f(x,y)$ 在区域 $D$ 内有偏导数
        一阶偏导
        >$$\frac { \partial z } { \partial x } = f _ { x } ( x , y ) , \quad \frac { \partial z } { \partial y } = f _ { y } ( x , y ) $$

        二阶偏导
        >$$\frac { \partial } { \partial x } ( \frac { \partial z } { \partial x } ) = \frac {\partial ^{2} z}{\partial x^{2}} = f_{xx}(x,y) , \frac { \partial } { \partial y } ( \frac { \partial z } { \partial x } ) = \frac {\partial ^{2} z}{\partial x \partial y} = f_{xy}(x,y) $$

        **拉普拉斯方程**，设函数为 $u=f(x,y,z)$，则存在
        >$$\frac { \partial ^ { 2 } u } { \partial x ^ { 2 } } + \frac { \partial ^ { 2 } u } { \partial z ^ { 2 } } + \frac { \partial ^ { 2 } u } { \partial z ^ { 2 } } = 0  $$
        
3. **全微分**
    * **定义**
        设函数 $z=f(x,y)$ 在点 $P(x,y)$ 的某邻域内有定义，函数在点 $P(x,y)$ 的全增量 $\Delta z = f ( x + \Delta x , y + \Delta y ) - f ( x , y ) $，可表示为
        >$$\Delta z = A \Delta x + B \Delta y + o ( \rho )\ \ \ (\rho = \sqrt { ( \Delta x ) ^ { 2 } + ( \Delta y ) ^ { 2 } })$$
        
        如果函数 $z=f(x,y)$ 再点 $(x,y)$ 可微分，根据增量与微分的关系得到的公式如下，全微分记作 $dz$
        >$$d z = A \Delta x + B \Delta y = A d x + B d y  $$

        $A$ 和 $B$ 表示点 $(x,y)$ 的偏导数
        \
        注意！多元函数在某点的偏导数存在，并不能保证在该点连续。但是如果函数在该点可微分，那么函数在该点必定连续
        \
        全微分存在的必要条件：
        如果 $z=f(x,y)$ 在点 $(x,y)$ 可微分，则偏导数 $\frac {\partial z}{\partial x}$ $\frac {\partial z}{\partial y}$ 必定存在，且函数在该点的全微分为
        >$$d z = \frac { \partial z } { \partial x } \Delta x + \frac { \partial z } { \partial y } \Delta y = \frac { \partial z } { \partial x } d x + \frac { \partial z } { \partial y } d y$$

        全微分存在的充分条件：
        如果函数 $z=f(x,y)$ 的偏导数 $\frac {\partial z} {\partial x}$ $\frac {\partial z} {\partial y}$ 在点 $(x,y)$ 连续，则函数在该点可微分

    * **全微分近似计算**
        >$$\Delta z \approx d z = f _ { x } ( x , y ) \Delta x + f _{y} ( x , y ) \Delta y  $$

        >$$f ( x + \Delta x , y + \Delta y ) \approx f ( x , y ) + f _ { x } ( x , y ) \Delta x + f _{y} ( x , y ) \Delta y $$

        上述两式可做近似计算
        例如：求 $1.04^{2.02}$ 的近似值
        >上述函数可看成 $z=f(x,y)=x^{y}$，取 $x=1, y=2, \Delta x=0.04, \Delta y=0.02$
        >$f_{x}(x,y) = yx^{y-1}, f_{y}(x,y) = x^{y}\ln x$
        >$1.04^{2.02} = (1+0.04)^{2+0.02} = 1^{2} + f_{x}(x,y)0.04 + f_{y}(x,y)0.02 = 1.08$

4. **多元复合函数求导**
    * **一元与多元函数复合的情景**
        如果函数 $u = \varphi ( t )$ 和 $v = \psi ( t ) $ 在点 $t$ 可导，函数 z=f(u,v) 在对应点 (u,v) 具有连续偏导数，则
        >$$\frac { d z } { d t } = \frac { \partial z } { \partial u } \frac { d u } { d t } + \frac { \partial z } { \partial v } \frac { d v } { d t } $$

    * **多元与多元函数复合的情景**
        * **偏导形式**
            如果函数 $u = \varphi ( x,y )$ 和 $v = \psi ( x,y ) $ 在点 $(x,y)$ 可导，函数 $z=f(u,v)$ 在对应点 $(u,v)$ 具有连续偏导数
            >$$ \frac { \partial z } { \partial x } = \frac { \partial z } { \partial u } \frac { \partial u } { \partial x }  + \frac { \partial z } { \partial v } \frac { \partial v } { \partial x }  $$

            >$$ \frac { \partial z } { \partial y } = \frac { \partial z } { \partial u } \frac { \partial u } { \partial y }  + \frac { \partial z } { \partial v } \frac { \partial v } { \partial y }  $$

        * **全微分形式**
            函数 $z=f[\varphi ( x,y ), \psi ( x,y )]$ 存在 (x,y) 的全微分
            >$$d z = \frac { \partial z } { \partial x } d x + \frac { \partial z } { \partial y } d y $$

            >$$d z = (\frac { \partial z } { \partial u } \frac { \partial u } { \partial x }  + \frac { \partial z } { \partial v } \frac { \partial v } { \partial x } )d x + (\frac { \partial z } { \partial u } \frac { \partial u } { \partial y }  + \frac { \partial z } { \partial v } \frac { \partial v } { \partial y })d y $$

    * **链式法则**
        若 $z=f(u_{1},\cdots,u_{m})$ 在点 $(u_{1},\cdots,u_{m})$ 可微，$u_{i}=g_{i}(x_{1},\cdots,x_{n}) \ (i=1,\cdots,m)$ 在点 $(x_{1},\cdots,x_{n})$ 有关于 $x_{j} \ (j=1,\cdots,n)$ 的偏导数，则复合函数：
        >$$z=f[g_{1}(x_{1},\cdots,x_{n}),\cdots,g_{m}(x_{1},\cdots,x_{n})]$$

        关于自变量 $x_{j}$ 的偏导为：
        >$$\frac {\partial z}{\partial x_{j}} = \sum\limits_{i=1}^{m} \frac {\partial f}{\partial u_{i}} \cdot \frac {\partial u_{i}}{\partial x_{j}} \ \ \ (j=1,\cdots,n)$$

        写成矩阵的形式
        >$$(\frac {\partial z}{\partial x_{1}}, \cdots, \frac {\partial z}{\partial x_{n}}) = (\frac {\partial f}{\partial u_{1}}, \cdots, \frac {\partial f}{\partial u_{m}}) \left(\begin{array}{l} \frac {\partial u_{1}}{\partial x_{1}} & \cdots & \frac {\partial u_{1}}{\partial x_{n}} \\ \vdots & & \vdots \\ \frac {\partial u_{m}}{\partial x_{1}} & \cdots & \frac {\partial u_{m}}{\partial x_{n}} \end{array}\right)$$

    * **雅可比矩阵**
        设 $f:R_{n} \rightarrow R_{m}$ 是一个函数，它的输入向量 $x \in R_{n}$，它的输出变量 $y = f(x) \in R_{m}$
        >$$\left\lbrace\begin{array}{l} { y_{1}=f_{1}(x_{1},\cdots,x_{n}) } \\ { y_{2}=f_{2}(x_{1},\cdots,x_{n}) } \\ {\cdots} \\ {y_{m}=f_{1}(x_{1},\cdots,x_{n}) } \end{array}\right. $$

        **雅可比矩阵**是一个 $m×n$ 矩阵
        >$$J = [\frac {\partial f}{\partial x_{1}} \cdots \frac {\partial f}{\partial x_{n}}] = \left[\begin{array}{l} \frac {\partial f_{1}}{\partial x_{1}} & \cdots & \frac {\partial f_{1}}{\partial x_{n}} \\ \vdots & & \vdots \\ \frac {\partial f_{m}}{\partial x_{1}} & \cdots & \frac {\partial f_{m}}{\partial x_{n}} \end{array}\right]$$

        此矩阵表示为
        >$$J_{F(x_{1},\cdots,x_{n})} = \frac {\partial (y_{1},\cdots,y_{m})}{\partial (x_{1},\cdots,x_{n})}$$

        雅可比矩阵可看成 $R_{n} (x_{1} \cdots x_{n}) \rightarrow R_{m}(y_{1} \cdots y_{m})$ 是一个从n维欧氏空间转换到m维欧氏空间的函数
        \
        如果 $p(x_{1},\cdots,x_{n})$ 是 $R_{n}$ 中的一个点，$F$ 在 $p$ 点可微分，那么这一点的导数由 $J_{F(p)}$ 给出，在此情况下，由 $J_{F(p)}$ 描述的线性算子即接近点 $p$ 的 $F$ 的最优线性逼近，$x$ 逼近于 $p$
        >$$F(x) \approx F(p) + J_{F(p)}(x-p)  $$

    * **雅可比行列式**
        当 $m = n$ 时，可以定义雅可比矩阵 $J$ 的行列式，在微积分换元中，也就是给出了从 $x$ 到 $y$ 的 $n$ 维体积的比率
        >$$d y_{1} \cdots d y_{n} = |J| d x_{1} \cdots d x_{n} $$

        如果连续可微函数f在p点的雅可比行列式不是零，那么它在该点附近具有反函数。


5. **隐函数求导**
    * **一个方程的情景**
        设函数 $F(x,y)$ 在点 $(x_{0},y_{0})$ 的某一邻域内满足：
        >(1) 具有连续的偏导数
        >(2) $F(x_{0}, y_{0}) = 0$
        >(3) $F_{y}(x_{0}, y_{0}) \neq 0$ (函数 $F$ 在点 $(x_{0}, y_{0})$ 处对 $y$ 的偏导数)

        则方程 $F(x,y)$ 在点 $x_{0}$ 的某邻域内可确定一个函数 y=f(x) 具有如下性质：
        >(1) $y_{0}=f(x_{0})$
        >(2) 在 $x_{0}$ 的上述邻域内连续
        >(3) 在 $x_{0}$ 的上述邻域内可导，且有
        >$$\frac {dy}{dx} = - \frac {F_{x}} {F_{y}} \ \ \ (F_{x} 和 F_{y} 表示 F 对 x 和 y 求偏导)$$

        推导
        >$F(x,y)=F(x,f(x)) = 0$ 可看成是 $x$ 的复合函数，求导
        >$\frac { \partial F } { \partial x } + \frac { \partial F } { \partial y } \frac { d y } { d x } = 0 $
        >由于 $F_{y}$ 连续，且 $F_{y}(x_{0}, y_{0}) \neq 0$，所以存在 $(x0,y0)$ 一个邻域，使得 $F_{y} \neq 0$
        >$\frac {dy}{dx} = - \frac {F_{x}} {F_{y}}$

        设函数 $F(x,y,z)$ 在点 $(x_{0},y_{0},z_{0})$ 的某一邻域内满足：
        >(1) 具有连续的偏导数
        >(2) $F(x_{0}, y_{0}, z_{0}) = 0$
        >(3) $F_{y}(x_{0}, y_{0}, z_{0}) \neq 0$

        则方程 $F(x,y,z)$ 在点 $(x_{0}, y_{0})$ 的某邻域内可确定一个函数 z=f(x,y) 具有如下性质：
        >(1) $z_{0}=f(x_{0},y_{0})$
        >(2) 在 $(x_{0}, y_{0})$ 的上述邻域内连续
        >(3) 在 $(x_{0}, y_{0})$ 的上述邻域内可导，且有
        >$$\frac {\partial z}{ \partial x} = - \frac {F_{x}} {F_{z}} \ \ \ \frac {\partial z}{ \partial y} = - \frac {F_{y}} {F_{z}}$$
 

    * **方程组的情景**
        两个方程确定两个隐函数
        >$$\left\lbrace\begin{array}{l} { F ( x , y , u , v ) = 0 } \\ { G ( x , y , u , v ) = 0 } \end{array}\right. \ \ \ 其中 \left\lbrace\begin{array}{l} { u = u(x,y) } \\ { v=v(x,y) } \end{array}\right. $$

        由 $F,G$ 偏导组成的行列式，称 $F,G$ 的**雅可比行列式**
        >$$J = \frac {\partial (F,G)} {\partial (u,v)} = \left|\begin{array}{l} F_{u} & F_{v} \\ G_{u} & G_{v} \end{array}\right| = \left|\begin{array}{l} \frac {\partial F}{\partial u} & \frac {\partial F}{\partial v} \\ \frac {\partial G}{\partial u} & \frac {\partial G}{\partial v} \end{array}\right|$$

        设函数 $F(x,y,u,v)$ 和 $F(x,y,u,v)$满足：
        >(1) 在点 $P(x_{0},y_{0},u_{0},v_{0})$ 的某一邻域内具有连续偏导
        >(2) $F(x_{0},y_{0},u_{0},v_{0}) = 0$，$G(x_{0},y_{0},u_{0},v_{0}) = 0$
        >(3) $J |_{P} = \frac {\partial (F,G)} {\partial (u,v)} |_{P} \neq 0$
        
        则方程组 $F(x,y,u,v)=0$，$G(x,y,u,v)=0$ 在点 $(x_{0}, y_{0})$ 的某邻域内可确定一组满足条件 $u_{0}=u(x_{0},y_{0})$，$v_{0}=v(x_{0},y_{0})$ 的连续函数 $u=u(x,y)$，$v=v(x,y)$ 具有如下偏导公式：
        >$$\frac { \partial u } { \partial x } = - \frac { 1 }{ J } \frac {\partial (F,G)}{\partial (x,v)} \ \ \ \ \frac { \partial u } { \partial y } = - \frac { 1 }{ J } \frac {\partial (F,G)}{\partial (y,v)} \ \ \ \ \ (J \neq 0)  $$

        >$$\frac { \partial v } { \partial x } = - \frac { 1 }{ J } \frac {\partial (F,G)}{\partial (u,x)} \ \ \ \ \frac { \partial v } { \partial y } = - \frac { 1 }{ J } \frac {\partial (F,G)}{\partial (u,y)} \ \ \ \ \ (J \neq 0)$$


6. **多元函数微分学的几何应用**
    * **空间曲线的切线与法平面**
        设空间曲线的参数方程为：$\left\lbrace\begin{array}{l} { x = \varphi ( t ) } \\ { y = \psi ( t ) } \\ { z = \omega ( t ) } \end{array}\right.  \ \ \ t \in \lbrack \alpha , \beta \rbrack  $
        
        写成向量的形式：$f(t) = \varphi ( t ) i + \psi ( t ) j + \omega ( t ) k$
        曲线上的每一点可表示为：$f(t)=(\varphi ( t ) + \psi ( t ) + \omega ( t )), t \in [\alpha,\beta]$

        在曲线 $\Gamma$ 上与点 $M$ 对应的参数 $t_{0}$，向量 $T=f^{\prime}(t_{0})=(\varphi ^ { \prime } ( t _ { 0 } ), \psi ^ { \prime } ( t _ { 0 } ) , \omega ^ { \prime } ( t _ { 0 } ))$ 表示曲线 $\Gamma$ 在 $M$ 点的切向量，其**切线方程**为：
        >$$\frac { x - x _ { 0 } } { \varphi ^ { \prime } ( t _ { 0 } ) } = \frac { y - y _ { 0 } } { \psi ^ { \prime } ( t _ { 0 } ) } = \frac { z - z _ { 0 } } { \omega ^ { \prime } ( t _ { 0 } ) }  $$

        通过点M且与切线垂直的平面称为曲线 $\Gamma$ 在点 $M$ 处的**法平面方程**为：
        >$$\varphi ^ { \prime } ( t _ { 0 } ) ( x - x _ { 0 } ) + \psi ^ { \prime } ( t _ { 0 } ) ( y - y _ { 0 } ) + \omega ^ { \prime } (t_{0} )( z-z _ { 0 } ) = 0  $$

    * **空间曲面的切平面与法线**
        曲平面方程：$F(x,y,z)=0$
        \
        设 $M(x_{0},y_{0},z_{0})$ 是曲面上的一点。通过点 $M$ 在曲面上引一条曲线 $\Gamma$ ，其参数方程为 $x=\varphi(t),y=\psi(t),z=\omega(t)$
        \
        $t=t_{0}$ 对应于点 $M(x_{0},y_{0},z_{0})$ 且 $\varphi ^ { \prime } ( t _ { 0 } ), \psi ^ { \prime } ( t _ { 0 } ), \omega ^ { \prime } ( t _ { 0 } )$ 不全为零，此曲线的切线为：$\frac { x - x _ { 0 } } { \varphi ^ { \prime } ( t _ { 0 } ) } = \frac { y - y _ { 0 } } { \psi ^ { \prime } ( t _ { 0 } ) } = \frac { z - z _ { 0 } } { \omega ^ { \prime } ( t _ { 0 } ) }  $
        \
        因为曲线在曲面上，所以 $F[\varphi (t), \psi (t), \omega (t)] = 0$，因为在点 $M(x_{0},y_{0},z_{0})$ 处有连续的偏导数，所以 $t=t_{0}$ 的全导数为
        >$\frac {d}{dt} F[\varphi (t), \psi (t), \omega (t)] |_{t=t_{0}} = 0$
        >根据全微分定义可写成
        >$F _ { x } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) \varphi ^ { \prime } ( t _ { 0 } ) + F _ { y } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) \psi ^{\prime} ( t_{0} ) + F _ { z } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) \omega ^{\prime} ( t_{0} ) = 0  $
        >引入向量 $n = (F _ { x } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) , F _ { y } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) , F _ { z } ( x _ { 0 } , y _ { 0 } , z _ { 0 } )  )$
        >点 $M$ 的切向量 $(\varphi ^ { \prime } ( t _ { 0 } ) , \psi ^ { \prime } ( t_{0} ) , \omega ^ { \prime } ( t_{0} ))$
        >向量 $n$ 垂直于点 $M$ 处所有的切线，所以点 $M$ 处的**切平面**为：

        >$$F _ { x } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) ( x - x _ { 0 } ) + F _ { y } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) ( y - y _ { 0 } ) + F _ {z} ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) ( z - z _ { 0 } ) = 0 $$

        通过其切平面的**法线**为：
        >$$\frac { ( x - x _ { 0 } ) } { F _ { x } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) } = \frac { ( y - y _ { 0 } ) } { F _ { y } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) } = \frac { ( z - z _ { 0 } ) } { F _ { z } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) } $$

        **法向量**：$(F _ { x } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) ,F _ { y } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ), F _ { z } ( x _ { 0 } , y _ { 0 } , z _ { 0 } ) )$

7. **方向导数与梯度**
    * **方向导数**
        如果函数 $f(x,y)$ 在点 $P_{0}(x_{0},y_{0})$ 可微分，那么函数在该点沿任一方向 $l$ 的方向导数存在，且有：
        >$$\frac {\partial f}{\partial l} |_{(x_{0},y_{0})} = f_{x}(x_{0},y_{0})\cos \alpha + f_{y}(x_{0},y_{0})\cos \beta$$

        $\cos \alpha, \cos \beta$ 是方向 $l$ 的方向余弦
        \
        同理对于 $f(x,y,z)$ 在点 $(x_{0},y_{0},z_{0})$ 的方向导数为
        >$$\frac {\partial f}{\partial l} |_{(x_{0},y_{0},z_{0})} = f_{x}(x_{0},y_{0},z_{0})\cos \alpha + f_{y}(x_{0},y_{0},z_{0})\cos \beta + f_{z}(x_{0},y_{0},z_{0})\cos \gamma$$

    * **梯度**
        设函数 $f(x,y)$ 在平面区域 $D$ 内具有一阶连续偏导，则对于每一点 $P_{0}(x_{0},y_{0}) \in D$，存在如下向量，被称为函数 $f(x,y)$ 在点 $P_{0}(x_{0},y_{0})$ 的**梯度**，记为 $grad \ f(x_{0},y_{0})$ 或 $\nabla f(x_{0},y_{0})$，$(i,j)$ 表示单位向量
        >$$\nabla f(x_{0},y_{0}) = f_{x}(x_{0},y_{0})i + f_{y}(x_{0},y_{0})j$$

        **函数在某点的梯度是一个向量，它的方向与取得最大方向导数的方向一致，它的模为方向导数的最大值**
        \
        其中 $\nabla = \frac {\partial}{\partial x}i + \frac {\partial}{\partial y}j $ 称为**向量微分算子**或**Nabla算子**，$\nabla f = \frac {\partial f}{\partial x}i + \frac {\partial f}{\partial y}j $
        \
        梯度与方向导数的关系：如果函数 $f(x,y)$ 在点 $p_{0}(x_{0},y_{0})$ 可微分，$e_{l}=(\cos \alpha, \cos \beta)$ 是与方向 $l$ 同向的单位向量
        >$$\frac {\partial f}{\partial l}|_{x_{0},y_{0}} = f_{x}(x_{0},y_{0}) \cos \alpha + f_{y}(x_{0},y_{0}) \cos \beta $$
        
        >$$ = grad \ f(x_{0}, y_{0}) \cdot e_{l} = |grad \ f(x_{0}, y_{0})| \cos \theta $$

        (1) 当 $\theta = 0$，即方向 $e_{l}$ 与梯度 $grad \ f(x_{0},y_{0})$ 的方向相同，函数 $f(x,y)$ 的增加最快。此时函数在这个方向的方向导数达到最大值，即梯度的模
        >$\frac {\partial f}{\partial l}|_{x_{0},y_{0}} = |grad \ f(x_{0}, y_{0})|$

        (2) 当 $\theta = \pi$，即方向 $e_{l}$ 与梯度 $grad \ f(x_{0},y_{0})$ 的方向相反，函数 $f(x,y)$ 的减小最快。函数在这个方向的方向导数达到最小值，即
        >$\frac {\partial f}{\partial l}|_{x_{0},y_{0}} = -|grad \ f(x_{0}, y_{0})|$

        (3) 当 $\theta = \frac {\pi}{2}$，即方向 $e_{l}$ 与梯度 $grad \ f(x_{0},y_{0})$ 的方向正交(垂直)，函数 $f(x,y)$ 的变化为零
        >$\frac {\partial f}{\partial l}|_{x_{0},y_{0}} = |grad \ f(x_{0}, y_{0})| \cos \theta = 0$
        
        \
        函数 $f(x,y,z)$ 在点 $(x_{0},y_{0},z_{0})$ 的梯度 $\nabla f(x_{0},y_{0},z_{0})$ 的方向就是等值线 $f(x,y,z)=c$ 在这点的法线方向 $n$，梯度的模 $|\nabla f(x_{0},y_{0},z_{0})|$ 就是函数沿法线方向的方向导数 $\frac {\partial f}{\partial n}$：
        一个点的**梯度**
        >$$\nabla f(x_{0},y_{0},z_{0})$$

        **梯度的方向**（等值线在这点法线的方向）
        >$$n=\frac {\nabla f(x_{0},y_{0},z_{0})}{|\nabla f(x_{0},y_{0},z_{0})|}$$

        法线方向的**方向导数**
        >$$\frac {\partial f}{\partial l}|_{x_{0},y_{0},z_{0}} = |grad \ f(x_{0}, y_{0}, z_{0})|$$

8. **二元函数的极值与求法**
    * **二元函数的极值**
        （必要条件）如果函数 $z=f(x,y)$ 在点 $(x_{0},y_{0})$ 处有偏导数，且在点 $(x_{0},y_{0})$ 有极值，则该点的偏导数必然为零：
        >$$f_{x}(x_{0},y_{0})=0 \ \ \ f_{y}(x_{0},y_{0})=0$$

        >$$\nabla f(x_{0},y_{0}) = 0$$

        （充分条件）函数 $z=f(x,y)$ 在点 $(x_{0},y_{0})$ 的某邻域内连续且有一阶及二阶的连续偏导数，又 $f_{x}(x_{0},y_{0})=0$，$f_{y}(x_{0},y_{0})=0$，令
        >$$f_{xx}(x_{0},y_{0})=A \ \ \ f_{xy}(x_{0},y_{0})=B \ \ \ f_{yy}(x_{0},y_{0})=C$$

        极值条件如下：
        >$AC-B^{2}>0$ 时具有极值，且 $A<0$ 时有极大值，$A>0$ 时有极小值
        
        >$AC-B^{2}<0$ 时没有极值
        
        >$AC-B^{2}=0$ 时，可能有极值，也可能没有极值

        求极值的一般解法如下：
        （1）解方程组 $f_{x}(x_{0},y_{0})=0$，$f_{y}(x_{0},y_{0})=0$，会得到一些驻点
        （2）对于每一个驻点 $(x_{0},y_{0})$ 求出 $A，B和C$ 的值
        （3）定出 $AC-B^{2}$ 的符号，判断是否有极值，极大还是极小

    * **条件极值 拉格朗日乘法**
        上述讨论的极值问题属于自变量无条件的极值问题，此处针对自变量有条件的极值问题
        函数 $z=f(x,y)$ 在附加条件 $\varphi(x,y)=0$ 下存在极值点，可以先做**拉格朗日函数**
        >$$L(x,y) = f(x,y)+ \lambda \varphi(x,y)$$
        
        参数 $\lambda$ 为拉格朗日算子。上述函数对 x y的一阶偏导，并使之为零，得到方程组：
        >$$\left\lbrace\begin{array}{l} { f _ { x } ( x , y ) + \lambda \varphi _ { x } ( x , y ) = 0 } \\ { f_{y} ( x , y ) + \lambda \varphi _ { y } ( x , y ) = 0 } \\ { \varphi ( x , y ) = 0 } \end{array}\right.  $$

        如果存在多个条件，那么：
        >$$L(x,y,z,t) = f(x,y,z,t)+ \lambda \varphi(x,y,z,t) + u \psi(x,y,z,t)$$

        推导
        >函数：$z=f(x,y)$，条件：$\varphi(x,y)=0$
        >设 $y=\psi(x)$，那么 $z=f[x, \psi(x)]$
        >如果在 $(x_{0}, y_{0})$ 存在极值，也就是相当于函数 $z=f[x, \psi(x)]$ 在 $x=x_{0}$ 的极值，如下（一元与多元函数复合的情景）
        >$\frac {dz}{dx}|_{x=x_{0}} = f_{x}(x_{0},y_{0}) + f_{y}(x_{0},y_{0})\frac {dy}{dx}|_{x=x_{0}} = 0$
        >根据隐函数求导公式：$\frac {dy}{dx}|_{x=x_{0}}$ = - \frac {\varphi_{x}(x_{0},y_{0})}{\varphi_{y}(x_{0},y_{0})}
        >$f_{x}(x_{0},y_{0}) - f_{y}(x_{0},y_{0}) \frac {\varphi_{x}(x_{0},y_{0})}{\varphi_{y}(x_{0},y_{0})} = 0$
        >同理，设 $x=\psi(y)$，相当于函数 $z=f[\psi(y), y]$ 在 $y=y_{0}$ 的极值


8. **泰勒公式与极值**
    * **二元函数的泰勒公式**
        函数 $f(x)$ 在开区间 $(a,b)$ 内具有 $(n+1)$ 阶可导，则一元泰勒公式：
        >$$f ( x ) = f ( x _ { 0 } ) + f ^ { \prime } ( x _ { 0 } ) ( x - x _ { 0 } )  $$

        >$$+ \frac { f ^ { \prime \prime } ( x _ { 0 } ) } { 2 } ( x - x _ { 0 } ) ^ { 2 } + \cdots + \frac { f ^ { (n) } ( x _ { 0 } ) } { n ! } ( x - x _ { 0 } ) ^ { n }  $$

        >$$+ \frac { f ^ { ( n + 1 ) } ( x _ { 0 } + \theta ( x - x _ { 0 } ) ) } { ( n + 1 ) ! } ( x - x _ { 0 } ) ^ { n + 1 } \quad ( 0 < \theta < 1 )  $$

        函数 $f(x,y)$ 在点 $(x_{0}, y_{0})$ 的某一邻域内连续且有直到 $(n+1)$ 阶的连续偏导，$(x_{0}+h, y_{0}+k)$ 为此邻域内任一点，则有**二元泰勒公式**
        >$$f ( x _ { 0 } + h , y _ { 0 } + h ) = f ( x _ { 0 } , y _ { 0 } ) + ( h \frac { \partial } { \partial x } + k \frac { \partial } { \partial y } )f ( x _ { 0 } , y _ { 0 } ) $$

        >$$+\frac { 1 } { 2! } ( h \frac { \partial } { \partial x } + k \frac { \partial } { \partial y } ) ^ { 2 } f ( x _ { 0 } , y _ { 0 } ) + \cdots + \frac { 1 } { n! } ( h \frac { \partial } { \partial x } + k \frac { \partial } { \partial y } ) ^ { n } f ( x _ { 0 } , y _ { 0 } ) + R_{n}$$

        上述公式称为二元函数 $f(x,y)$ 在点 $(x_{0},y_{0})$ 的 $n$ 阶泰勒公式，其中
        >$$R_{n} = \frac {1}{(n+1)!} (h \frac{\partial}{\partial x} + k \frac{\partial}{\partial y})^{n+1} f(x_{0}+\theta h, y_{0}+\theta k)  \ \ \ ( 0 < \theta < 1 )$$

        $R_{n}$ 称为**拉格朗日型余项**

        当 $n = 0$ 时，下式称为**二元函数的拉格朗日中值公式**
        >$$f(x_{0}+h, y_{0}+k) = f(x_{0}, y_{0}) + hf_{x}(x_{0}+\theta h, y_{0}+\theta k) + kf_{y}(x_{0}+\theta h, y_{0}+\theta k)$$

    * **多元二阶泰勒公式**
        定义一个开区域 $D \in R^{n}$ 上的 $n$ 元二次可微函数 $f(x_{1},\cdots,x_{n})$，设 $p_{0}=(x_{1}^{0},\cdots,x_{n}^{0}) \in D$，其二阶泰勒展开可写成：
        >$$f(p_{0}+ \Delta x) = f(p_{0}) + \nabla f(p_{0}) \cdot \Delta x^{T} + \frac {1}{2!} \Delta x Q \Delta x^{T} + o(r^{2})$$

        其中 
        >$\Delta x = (\Delta x_{1},\cdots,\Delta x_{n}) = (x_{1}-x_{1}^{0},\cdots,x_{n}-x_{n}^{0})$
        >$\nabla f(p_{0}) 是 f(x) 在 p_{0} 的梯度$
        >$Q = (\frac {\partial ^{2} f}{\partial x_{i} \partial x_{j}})|_{x=p_{0}}$ 称为 **$Hesse$ 矩阵**

    * **Hesse(海森矩阵)**
        如果多元函数 $f(x_{1},\cdots,x_{n})$ 所有的二阶导数都存在，那么 $f$ 的Hesse(海森矩阵)即：$H(f)_{ij}(x) = D_{i}D_{j}f(x)$
        >$$H(f) = \left|\begin{array}{l} \frac { \partial ^ { 2 } f } { \partial x _ { 1 } ^ { 2 } } & \frac { \partial ^ { 2 } f } { \partial x _ { 1 } \partial x _ { 2 } } & \cdots & \frac { \theta ^ { 2 } f } { \partial x _ { 1 } \partial x _ { n } }   \\ \frac { \partial ^ { 2 } f } { \partial x _ { 2 } \partial x _ { 1 } } & \frac { \partial ^ { 2 } f } { \partial x _ { 2 } ^{2} } & \cdots & \frac { \theta ^ { 2 } f } { \partial x _ { 2 } \partial x _ { n } }  \\ \vdots & \vdots & \ddots & \vdots \\ \frac { \partial ^ { 2 } f } { \partial x _ { n } \partial x _ { 1 } } & \frac { \partial ^ { 2 } f } { \partial x _ { n } \partial x _ { 2 } } & \cdots & \frac { \theta ^ { 2 } f } { \partial x _ { n }^{2} }    \end{array}\right|$$

        海森矩阵被应用于牛顿法解决的大规模优化问题

    * **极值**
        对于多元二次连续可微函数 $f(x_{1},\cdots,x_{n})$，设 $p_{0}=(x_{1}^{0},\cdots,x_{n}^{0})$ 是函数的极值点，那么 $\partial f(p_{0})=0$，由二阶泰勒展开得：
        >$$f(p_{0}+\Delta x) = f(p_{0}) + \frac {1}{2!} \Delta x Q \Delta x^{T} + o(r^{2})$$

        根据二次型 $\Delta x Q \Delta x^{T}$ 的符号 来确定 $f(x)$ 是否在 $p_{0}$ 处取得极值
        >若 $Q$ 是正定的，则 $f(p_{0})$ 是极小值

        >若 $Q$ 是负定的，则 $f(p_{0})$ 是极大值

        >若 $Q$ 是不定的，则 $f(p_{0})$ 不是极值

        >若 $Q$ 是半定的（所有特征值同号，由零特征值），则需要进一步判断
